# 时间意识模块 (Temporal Consciousness)

## 理论基础

本模块基于 **Stanford Encyclopedia of Philosophy** 时间意识与现象学理论，整合了 Husserl、Heidegger、Merleau-Ponty 等现象学家的时间理论。

### 核心理论框架

#### 1. Husserl 时间意识三重结构

- **原初印象 (Primal Impression)**：当下的直接体验，时间意识的核心点
- **保留 (Retention)**：刚刚过去的体验在意识中的持存，不是回忆而是"活生生的过去"
- **预期 (Protention)**：对即将到来的体验的前摄性预期，不是预测而是"活生生的未来"

这三个维度构成了"活生生的现在"(Living Present)，使时间体验成为连续的流而非离散的点。

#### 2. 时间深度 (Temporal Depth)

个体心理时间视野的广度分为 5 个层次：
1. **瞬间导向**：只关注当下即刻
2. **短期导向**：数小时到数天
3. **中期导向**：数周到数月
4. **长期导向**：数年到数十年
5. **超越导向**：超越个人生命的时间范围

#### 3. 时间整合状态

- **时间整合**：过去 - 现在 - 未来和谐统一
- **过去固着**：被过去经历主导
- **未来焦虑**：过度担忧未来
- **现在分裂**：当下体验碎片化

### 核心概念

1. **时间性 (Temporality)**：存在的基本结构，不是外在的测量而是内在的体验
2. **时间连续体**：过去 - 现在 - 未来不是分离的，而是相互渗透的整体
3. **前反思自我意识**：在反思之前就已经存在的自我觉知
4. **具身时间**：时间体验通过身体得以实现

## 功能特性

### 练习模块

1. **时间锚定** - 建立过去 - 现在 - 未来的心理连续性
2. **预期想象** - 通过现象学想象构建积极的未来图景
3. **回忆整合** - 重新理解过去经历的时间意义
4. **当下临在** - 培养深度的现在感
5. **时间透视** - 扩展心理时间视野
6. **时间流日记** - 记录时间体验的流动

### 评估工具

- **时间整合状态评估** - 分析过去/现在/未来导向的平衡
- **时间深度评估** - 测量心理时间视野的广度

## 使用示例

```javascript
const TemporalConsciousness = require('./src/temporal-consciousness');

// 与用户交互
const response = TemporalConsciousness.interact('我总是担心未来会发生什么');
console.log(response.guidance);

// 评估时间整合状态
const assessment = TemporalConsciousness.assessIntegration('我总是在想过去的事情，很难放下');
console.log(assessment.state); // '过去导向'
console.log(assessment.suggestion);

// 评估时间深度
const depth = TemporalConsciousness.assessTemporalDepth('我一般只计划这周的事情');
console.log(depth.level); // '短期导向'
```

## 应用场景

- **时间焦虑**：帮助过度担忧未来的人回到当下
- **创伤后成长**：整合过去创伤经历的时间意义
- **生命意义探索**：通过时间透视寻找生命方向
- **正念练习**：深化当下临在感
- **决策支持**：用长远视角指导当下选择

## 与其他模块的关系

- **叙事心理学**：时间意识为生命故事提供时间维度
- **自我意识**：时间性是自我意识的核心结构
- **情绪理论**：情绪体验具有内在的时间结构
- **能动性**：行动意图依赖于预期结构
