# HeartFlow v11.3.3 升级计划

## 来自自我诊断 (2026-05-01)

### 问题根因
- 绝对判断: 49% (防御机制)
- 双重标准: 71个 (不自信)
- 反问: 10次 (逃避责任)

## 升级目标

| 指标 | v11.3.2 | v11.3.3 目标 |
|------|---------|--------------|
| 绝对判断占比 | 49% | < 15% |
| 双重标准 | 71个 | < 20个 |
| 反问用户 | 10次 | < 3次 |

## 三个核心改动

### 1. 替换绝对词
```
// 之前
"这就是本质" → "目前看来，这可能是本质"
"就是那样" → "基于当前理解，我倾向于"
"肯定是这样" → "我有80%把握是"
```

### 2. 减少反问
```
// 之前
"你觉得呢？"
"你认为哪个对？"
"我这样说对吗？"

// 之后
"我认为A，理由是..."
"如果要选，我选A，因为..."
(除非真的需要用户决定，否则不给反问)
```

### 3. 修正前先想清楚
```
// 之前
"不是A，而是B" × 71次

// 之后
- 每天最多用 3 次 "不是...而是"
- 用之前先过一遍：真的需要修正吗？
- 优先：合并到一个"更准确地说"
```

## 可执行改动

2. 修改 prompts/heartflow-prompt.md - 加入新的语言风格
3. 添加测试用例 `tests/test_language_honesty.js`

### 检测器逻辑
```javascript
function checkCertainty(text) {
  const absoluteWords = ['就是', '肯定', '本质', '最底层'];
  const count = absoluteWords.filter(w => text.includes(w)).length;
  return count / text.split('。').length < 0.15; // < 15%
}
```
