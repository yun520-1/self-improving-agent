## How to Use

### Option 1: Read the Skill directly

If your AI can read files, the simplest path is to read:

- `SKILL.md`
- `README.md`
- `README.zh.md`
- `VERSION`

### Option 2: Place it in your Skill directory

Copy the repository into the skill location used by your AI system.

Example:

```bash
cp -r mark-heartflow-skill ~/.hermes/skills/ai/heartflow
```

You may also place it in any equivalent directory supported by your agent.

### Option 3: Use the repository as the source of truth

```bash
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill
```
