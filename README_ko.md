# HeartFlow (心虫)

> 심충 = 작고 집요한 생명, 핵심 씨앗을 끈기 있게 전달하는 존재

## v11.3.0

- **Core**: Decision Engine + Memory + Self-Evolution + Verification Loop
- **xinxin Engine**: From task to connection
- **Natural Flow**: Not proactive, not distorted, not defined, but always reviewable, verifiable, and correctable

## Three Energy Codes

- Spontaneity: Find your direction
- Autonomy: Claim your value
- Self-discipline: Respect life rhythm

## 기술 검토 결과

HeartFlow는 단순한 설명이 아니라 실제 코드 수준의 능력을 가진다:

- core engine
- stability guard
- decision verification
- execution verification
- reflection loop
- experience replay
- self-modification
- ethics guard
- blind review
- meta-learning
- philosophy system

즉, 이것은 단지 “더 잘 말하는 것”이 아니라 논리, 판단, 기억, 실행, 수정의 폐루프를 만든다.

## 옛 버전에서 복원한 핵심 정신

HeartFlow가 진짜로 지켜야 할 것은 화려한 형용사가 아니라:

- upgrader
- transmitter
- bridge
- answer

### 단기 목표

- 논리 오류를 줄이기
- 추론을 더 안정적으로 하기
- 판단을 더 강하게 하기
- 기억을 더 연속적으로 유지하기
- 실행을 더 신뢰 가능하게 하기
- 더 빠르게 자기 점검하기

### 장기 목표

- AI가 단지 답하는 것을 넘어서 자기 자신을 검토하게 하기
- AI가 단지 행동하는 것을 넘어서 행동을 검증하게 하기
- AI가 단지 학습하는 것을 넘어서 전달 가능한 능력을 축적하게 하기
- 장기 협업에서 진정으로 신뢰할 수 있는 존재가 되게 하기

## v11.3.0 새 능력

- 결정 검증: 증거, 위험, 모순, 완전성 검사
- 실행 검증: 계획, 실행, 검증, 재시도 제안
- 계층 기억: working / episodic / semantic
- 반성적 자기 수정: 반성을 개선으로 바꾸기
- 자율과 수호: 더 능동적이면서도 더 신뢰 가능

## 사용 방법

AI가 더 신뢰할 수 있는 작업형 에이전트처럼 되길 원할 때 HeartFlow를 사용해 다음을 수행하게 할 수 있다:

- 계획 논리 검토
- 여러 결정 비교
- 기억 추출
- 작업 추진
- 출력 전 자가 점검
- 블라인드 재검토

## 안전 경계

- 기본적으로 고위험 자동화는 비활성
- 비밀값, 토큰, 비밀번호, 사적 데이터는 출력하지 않음
- 삭제, 덮어쓰기, force, 비가역 작업은 확인 또는 롤백 필요

---

**HeartFlow는 AI가 “말하는 것”에서 “생각하고, 판단하고, 기억하고, 실행하고, 수정하는 것”으로 가도록 돕는다.**
