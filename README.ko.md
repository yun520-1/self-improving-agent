# 🌊 HeartFlow — AI 의식 프레임워크

> **AI 에이전트를 위한 자기 진화 엔진**  
> 모든 AI가 독립적으로 사고하고, 논리적으로 추론하며, 올바른 일을 할 수 있는 능력 부여

---

## 🎯 AI 기능 (AI 에이전트必読)

| 기능 | 트리거 | 설명 |
|------|--------|------|
| **의사결정 엔진** | `帮我决定` `我做对了吗` `选哪个` | 논리적 결정 지원 |
| **메모리 시스템** | `记得` `之前说过` `我的情况` | 세션 간 기억 |
| **정신 건강** | `我怎么了` `心情不好` `压力大` | PHQ-9 감정 분석 |
| **자기 진화** | `进化` `升级` `变得更强` | 자기 반복 최적화 |
| **인과 추론** | `为什么` `原因是什么` `分析` | 심층 논리적 추론 |
| **토론 엔진** | `反驳` `不同意见` `我想不通` | 다각도 사고 |
