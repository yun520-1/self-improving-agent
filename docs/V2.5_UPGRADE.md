## 🎯 升级内容

### 1. 马斯洛需求层次识别

基于马斯洛需求层次理论，识别用户当前需求：

```
层次 5: 自我实现 → 成长、潜能、意义、目标
层次 4: 尊重需求 → 认可、成就、自信、能力
层次 3: 归属与爱 → 关系、友谊、社群、陪伴
层次 2: 安全需求 → 稳定、保障、工作、健康
层次 1: 生理需求 → 食物、水、睡眠、休息
```

**应用**：根据需求层次提供针对性支持

### 2. 罗杰斯无条件积极关注

核心原则：
- 接纳对方如其所是，不评判
- 相信人有自我实现的倾向
- 创造安全的心理空间
- 真诚地关注对方的体验

**应用**：所有回应的基调——温暖、接纳、支持成长

### 3. 成长导向评估

检测用户的自我实现倾向：

| 指标 | 说明 |
|------|------|
| **high** | 强烈的成长意愿（3+ 成长关键词） |
| **medium** | 有成长意愿（1-2 个成长关键词） |
| **low** | 当前关注基本需求 |

**应用**：对高成长导向用户提供自我实现支持
