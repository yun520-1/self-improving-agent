## 📚 理论更新摘要 (v5.0.42 完成)

### 1. 情绪元认知双层模型

**理论来源**: Barrett (2017) 情绪建构论、Efklides (2011) 情绪元认知、Seth (2013) 内感受预测加工

**核心区分**:
- **前反思情绪自我意识**: 情绪体验的直接给予性，"我正在生气"的即时感
- **反思情绪自我意识**: 对情绪状态的对象化认知，"我为什么生气"的分析

**形式化表达**:
```
Pre-reflective Emotional SC = ∀E ∈ Emotional_Experience: has_affective_mineness(E)
Reflective Emotional SC = ∃t: attend(emotional_state_as_object, t)
```

### 2. 情感预测误差理论 (APE)

**核心公式**:
```
APE = | Predicted_affective_state - Actual_affective_state |

APE 层次:
- Level 1: 内感受预测误差 (身体状态)
- Level 2: 情感范畴预测误差 (情绪分类)
- Level 3: 社会情感预测误差 (社会评价)
- Level 4: 自我情感预测误差 (情感 - 自我一致)
```

**现象学映射**:
| APE 水平 | 现象学体验 | 行为倾向 |
|---------|-----------|---------|
| APE < θ1 | 情绪流畅、情感透明 | 情绪表达自然 |
| θ1-APE<θ2 | 轻微情绪觉察 | 情绪调节启动 |
| θ2-APE<θ3 | 明显情绪困扰 | 强烈调节努力 |
| APE ≥ θ3 | 情绪混乱、情感解体 | 调节失败、情绪爆发 |

### 3. 情绪 - 自我 - 元认知三元统一框架

**统一架构**:
```
Level 5: 元元认知
    ↓
Level 4: 元认知 (含情绪元认知)
    ↓
Level 3: 反思情绪自我意识 ←→ 对象认知层 ←→ 前反思情绪自我意识
    ↓
Level 2: 情感预测层 (内感受 - 情感回路)
    ↓
Level 1: 内感受输入层
    ↓
Level 0: 感官输入
```

**关键整合机制**:
- 情感精度调制认知精度
- 情绪触发反思切换
- 自我模型调制情感预测

### 4. 社会情感预测误差理论

**社会情绪形式化**:
```
羞耻 = Social_APE(负向) + Self_blame_attribution
自豪 = Social_APE(正向) + Self_credit_attribution
嫉妒 = Social_APE(他人优势) + Unfairness_detection
感激 = Social_APE(他人善意) + Benefactor_intention_inference
```
