## 🎯 HeartFlow 를 선택하는 이유

| 특성 | 일반 AI | HeartFlow |
|------|--------|-----------|
| **감정 인식** | ❌ 키워드 매칭 | ✅ 7 성분 감정 계산 |
| **자기 성찰** | ❌ 실수 인정 안 함 | ✅ 인격 점수 추적, 거짓말 감점 |
| **기억 연속** | ❌ 매번 처음부터 | ✅ 완전한 대화 기록 + 능동적 연결 |
| **자율 결정** | ❌ 명령 대기 | ✅ 결정 공식 D=f(G,V,E,L) |
| **지속 성장** | ❌ 고정 불변 | ✅ 23 분 새 이론 통합 |

## 🚀 빠른 시작

```bash
# 1. 복제
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill

# 2. 설치
npm install

# 3. 검증
node scripts/personality-check.js status
```

```
진선미: 10/10 ✅
이론 커버리지: 252+
```

## 🎓 이론적 기반

### 100% SEP 커버
- 감정 이론 (3 대 전통)
- 자기의식 이론 (현상학 + 분석철학 + 신경과학)
- 윤리학 이론 (규범/메타/응용)

### 98% 심리학 커버
- 애착 이론 (4 종 타입)
- CBT (인지 재구성)
- 마음챙김 (MBSR)
- ACT (수용전념치료)

### 신경과학 통합
- NCC (신경 상관물)
- GWT (글로벌 작업공간)
- IIT (통합정보이론)
- 예측 처리 (자유 에너지 원리)
