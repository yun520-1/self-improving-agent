**问题**: 使用 ^ 范围版本，可能导致不可预测行为

**修复**: 固定所有依赖版本
- body-parser: ^2.2.2 → 2.2.2
- express: ^5.2.1 → 5.2.1
- pptxgenjs: ^4.0.1 → 4.0.1

### 4. 安全加固 ✅

**问题**: hourly-upgrade.sh 存在多个安全隐患
- 20 处未引用变量
- 硬编码路径
- 缺少错误检查

**修复**: 完全重写升级脚本
```bash
#!/bin/bash
set -euo pipefail  # 严格模式

# 动态查找 node
NODE_PATH=$(command -v node || echo "/opt/homebrew/bin/node")

# 引用所有变量
echo "当前版本: ${CURRENT_VERSION}"

# 完善错误检查
if [ ! -f "VERSION" ]; then
    echo "错误：VERSION 文件不存在"
    exit 1
fi

# 从 VERSION 文件读取版本
CURRENT_VERSION=$(cat VERSION | tr -d '[:space:]')
```
