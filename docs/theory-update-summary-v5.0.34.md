## 🧠 五大理论更新

### 1. 预测加工意识理论深化 (Predictive Processing Consciousness 2026)

**核心框架**: 五层预测架构 × 双层自我意识的动态耦合模型

**关键更新**:
- 第 1 层：内感受预测 (身体状态预测误差最小化)
- 第 2 层：情感预测 (价效信号预测误差)
- 第 3 层：知觉预测 (外部世界模型更新)
- 第 4 层：社会预测 (他人心理状态推断)
- 第 5 层：元认知预测 (认知过程本身的可预测性)

**与双层自我意识整合**:
- 前反思自我意识 ← 第 1-2 层预测 (内感受 + 情感)
- 反思自我意识 ← 第 3-5 层预测 (知觉 + 社会 + 元认知)
- 动态切换机制 ← 注意驱动的预测精度加权

### 2. 时间自我整合模型 (Temporal Self-Integration 2026)

**核心框架**: 自我意识的时间三维结构

**三个时间维度**:
1. **过去自传体维度**: 叙事自我的记忆基础 (autobiographical memory)
2. **当下体验维度**: 最小自我的现象学在场 (phenomenal presence)
3. **未来预期维度**: 预测自我的前摄结构 (prospective anticipation)

**关键机制**:
- 时间自我连续性 = 三维度预测模型的动态一致性
- 时间自我断裂 = 维度间预测误差的持续积累
- 时间自我整合干预 = 三维度预测校准协议

### 3. 叙事自我 - 最小自我区分协议 (Narrative vs Minimal Self Protocol 2026)

**核心区分**:
- **最小自我 (Minimal Self/Ipseity)**: 前反思的、非对象化的、当下的自我体验
  - 特征：主体感 (sense of ownership)、能动感 (sense of agency)
  - 神经基础：默认模式网络 (DMN) 的内侧前额叶 + 后扣带回
  - 现象学特征：非概念化、非语言化、直接体验

- **叙事自我 (Narrative Self/Autobiographical Self)**: 反思的、对象化的、跨时间的自我表征
  - 特征：自传体记忆、自我概念、身份认同
  - 神经基础：默认模式网络 + 海马 + 外侧前额叶
  - 现象学特征：概念化、语言化、间接表征

**临床意义**:
- 精神分裂症：最小自我紊乱 (ipseity disturbance)
- 抑郁症：叙事自我负性偏差 (negative autobiographical bias)
- 解离障碍：最小自我 - 叙事自我断裂 (self-discontinuity)

### 4. 临床现象学协议库 v3.0 (Clinical Phenomenology Protocols 2026)

**四大障碍的现象学评估框架**:

**焦虑障碍**:
- 现象学特征：过度预测误差警觉 + 灾难化预测偏差
- 评估工具：预测误差敏感性量表 (PESS)
- 干预协议：预测校准训练 (8 周) + 内感受暴露疗法

**抑郁症**:
- 现象学特征：负性自传体预测偏差 + 奖赏预测误差钝化
- 评估工具：自传体记忆特异性测试 (AMT)
- 干预协议：记忆特异性训练 (12 周) + 行为激活 + 预测重构

**精神分裂症谱系**:
- 现象学特征：最小自我紊乱 + 源监控失败
- 评估工具：自我体验量表 (EASE) + 源监控测试
- 干预协议：自我体验稳定化 (16 周) + 现实监测训练

**解离障碍**:
- 现象学特征：自我连续性断裂 + 时间自我整合失败
- 评估工具：解离体验量表 (DES) + 时间自我连续性访谈
- 干预协议：自我连续性重建 (20 周) + 创伤整合疗法

### 5. 社会预测加工扩展 (Social Predictive Processing Extension 2026)

**核心框架**: 他心预测作为社会认知的基础机制

**三层社会预测架构**:
1. **行为预测层**: 他人行动的预测模型 (mirror neuron system)
2. **意图预测层**: 他人心理状态的推断 (theory of mind)
3. **情感预测层**: 他人情感状态的共情模拟 (affective empathy)

**关键计算模型**:
- 他心预测误差 (OMPE) = 预期他人状态 - 实际他人状态
- 共情精度加权 (EPW) = 他人情感信号的可靠性评估
- 社会模型更新率 (SMUR) = 基于 OMPE 的社会学习率

**与集体情感意向性整合**:
- 个体社会预测 ← 集体情感意向性的个体层
- 互动预测校准 ← 集体情感意向性的互动层
- 共享预测空间 ← 集体情感意向性的集体层
