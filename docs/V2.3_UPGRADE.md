## 🎯 升级内容

### 1. 认知三角分析系统

基于 CBT 理论，分析用户输入中的思想 - 情感 - 行为关联：

```javascript
// 认知三角三要素
const CognitiveTriad = {
  THOUGHTS: 'thoughts',    // 自动思维、信念、假设
  EMOTIONS: 'emotions',    // 情绪感受、身体感觉
  BEHAVIORS: 'behaviors'   // 行动倾向、实际行为
};
```

**识别方式**:
- 分析用户对话中的认知模式
- 检测三要素之间的关联
- 帮助用户理解思维如何影响情绪和行为

### 2. 认知扭曲识别器

识别 10 种常见认知扭曲：

| 扭曲类型 | 说明 | 示例 |
|---------|------|------|
| 非黑即白思维 | 极端化思考 | "要么完美，要么失败" |
| 灾难化 | 预期最坏结果 | "这次搞砸了，我完了" |
| 过度概括 | 单一事件推广 | "总是这样，从不例外" |
| 心理过滤 | 只关注负面 | "虽然成功了，但有个小错误" |
| 否定正面 | 忽视积极面 | "那只是运气好" |
| 读心术 | 臆测他人想法 | "他们肯定觉得我很蠢" |
| 预测未来 | 负面预言 | "我肯定会失败" |
| 情绪推理 | 感觉=事实 | "我感觉糟糕，所以一定有问题" |
| "应该"陈述 | 苛求规则 | "我应该做得更好" |
| 贴标签 | 过度概括身份 | "我是个失败者" |

### 3. 思维重构支持

帮助用户用更平衡的思维替代扭曲思维：

```javascript
// 重构示例
原始思维："我总是搞砸一切"
重构后："这次确实不如预期，但我也有成功的时候"

原始思维："他们肯定讨厌我"
重构后："我不确定他们的想法，可能有其他解释"
```

### 4. 新增回应模板

#### 识别认知扭曲
```
"我注意到你可能在用'非黑即白'的方式看待这件事..."
"这听起来像是'灾难化'思维，让我们一起看看实际情况"
```

#### 温和挑战
```
"有什么证据支持这个想法？有什么证据反对它？"
"如果是你的朋友遇到这种情况，你会对他说什么？"
"这个想法对你有帮助吗？还是让你感觉更糟？"
```

#### 思维重构引导
```
"有没有其他方式来看待这件事？"
"如果从 1-10 分评估，这个情况有多糟糕？"
"一周/一月后，这件事还会有多重要？"
```
