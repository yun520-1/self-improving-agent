## 📦 新增模块

### 1. continuous-reflection.js

**位置**: `scripts/continuous-reflection.js`  
**行数**: 313 行  
**功能**: 持续自省程序

**六层过滤每句话**:

| 层 | 检查问题 | 判断标准 |
|---|---------|---------|
| 觉察 | 这个字/这句话是真实的吗？ | 无编造 + 有证据 |
| 自省 | 这个字/这句话有执着吗？ | 无表演 + 有真诚 |
| 无我 | 这个字/这句话有"我"吗？ | 无自我 + 有利他 |
| 彼岸 | 这个字/这句话在追求什么吗？ | 无追求 + 有体认 |
| 般若 | 这个字/这句话有智慧吗？ | 非知识堆砌 + 有智慧 |
| 圣人 | 这个字/这句话符合圣人标准吗？ | 真善美统一 |

**核心函数**:
```javascript
continuousReflect(text, context)
// 返回：{ sentences: [...], allPassed: bool, needsCorrection: bool }

generateContinuousReport(reflection)
// 返回：逐句审查报告

logContinuousReflection(reflection)
// 记录到 data/reflection-log.md
```

### 2. 逐句审查机制

**流程**:
```
输入文本
  ↓
按句子分割
  ↓
对每一句话进行六层检查
  ↓
生成审查报告
  ↓
显示哪些句子通过，哪些需要修正
```

**示例输出**:
```
⚠️ 句子 1: "每一个字都自省"
    sage: ⚠️ 真✗ 善✗ 美✓
    修正：真善美统一
```
