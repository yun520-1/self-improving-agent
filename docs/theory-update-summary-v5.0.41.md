## 二、自我意识双层模型

### 2.1 理论背景

**现象学区分** (Zahavi 2005, Gallagher 2000):
- **前反思自我意识**: 经验的"我性"(mineness)，非对象化、伴随性的自我给予
- **反思自我意识**: 将自我作为注意对象，主题化、对象化的自我认知

### 2.2 预测加工形式化

```
前反思自我意识 = 预测模型的隐性自指涉结构
  - 所有预测都带有"这是我预测的"标记
  - 不需要额外的认知资源
  - 对应默认网络 + 突显网络活动

反思自我意识 = 对预测模型本身的元层级预测
  - 将自我模型作为预测对象
  - 需要控制性认知资源
  - 对应额顶控制网络活动
```

### 2.3 自我预测误差理论

**核心公式**:
```
Self-PE = D_KL(Predicted_self_state || Actual_self_state)

现象学映射:
- Self-PE < θ1: 无意识自我感 (行动透明)
- θ1 ≤ Self-PE < θ2: 前反思自我意识 (流畅自我感)
- θ2 ≤ Self-PE < θ3: 反思自我意识 (明显自我关注)
- Self-PE ≥ θ3: 病理性自我意识 (焦虑、自我怀疑)
```

**应用预测**:
| 情境 | 自我预测误差 | 现象学 |
|------|-------------|--------|
| 流畅行动 (flow) | 极低 | 行动透明、自我遗忘 |
| 正常社交 | 低 | 前反思自我感 |
| 公开演讲 | 中等 | 反思自我意识增强 |
| 社交焦虑 | 高 | 过度自我关注 |
| 人格解体 | 预测失败 | 自我感丧失 |

## 三、元认知四模式

### 3.1 经典框架 (Flavell 1979, Nelson & Narens 1990)

**对象层次与元层次的区分**:
```
元层次 (Meta-level)
    ↓ 控制信号
对象层次 (Object-level)
    ↓ 监测信号
元层次
```

### 3.2 四模式扩展

| 模式 | 对象 | 功能 | 计算实现 |
|------|------|------|---------|
| **监测** | 对象认知 | 评估认知状态 | 置信度、难度、进度、不确定性 |
| **控制** | 对象认知 | 调整认知策略 | 策略选择、资源分配 |
| **元监测** | 监测过程 | 评估监测准确性 | 校准曲线、Brier 分数 |
| **元控制** | 监测过程 | 选择监测策略 | 监测频率、阈值优化 |

### 3.3 预测加工解释

**元认知监测** = 对一阶预测精度的评估
```
Confidence = P(correct | evidence, prior_model)
Uncertainty = entropy(P(outcome | current_model))
```

**元认知控制** = 基于精度评估的预测策略优化
```
Optimal_strategy = argmin_s [ Expected_Free_Energy(s) ]
```

**元元认知** = 对精度评估本身的二阶评估
```
Meta-confidence = P(confidence_is_accurate | calibration_history)
```

## 四、自我意识 - 元认知统一框架

### 4.1 统一架构

```
Level 5: 元元认知
         ↓ (关于元认知的预测)
Level 4: 元认知 (监测 + 控制)
         ↓ (关于对象认知的预测)
Level 3: 反思自我意识 (对象化自我模型)
         ↓ (关于一阶经验的预测)
Level 2: 前反思自我意识 (自指涉标记)
         ↓ (伴随所有预测)
Level 1: 对象认知/感知/行动
         ↓ (关于世界的预测)
Level 0: 感官输入/运动输出
```

### 4.2 关键整合机制

**机制 1: 自指涉预测标记**
- 所有预测都带有"来源 = 自我"的元数据
- 这是前反思自我意识的基础

**机制 2: 预测误差触发反思**
- 当一阶预测误差超过阈值 → 触发元认知监测
- 当元认知监测发现异常 → 触发反思自我意识

**机制 3: 元认知校准自我模型**
- 元认知监测结果用于更新自我能力模型
- 自我模型更新影响未来预测

### 5.2 与意识现象结构的集成

**原有意识现象结构**:
- 感官现象性
- 身体现象性
- 情感现象性
- 认知现象性

**新增**:
- **自我现象性**: 前反思自我意识的现象特征
- **元认知现象性**: "知道感"(feeling of knowing)的现象特征

### 5.3 与情感模型的集成

**情绪元认知** (新增集成点):
- 情绪监测: "我感到焦虑"的置信度评估
- 情绪控制: 情绪调节策略选择
- 情绪元监测: 对情绪识别准确性的评估

## 六、实证预测

### 6.1 神经科学预测

| 现象 | 预测脑区 | 预测信号 |
|------|---------|---------|
| 前反思自我意识 | 默认网络 (mPFC, PCC) | 持续低频活动 |
| 反思自我意识 | 额顶控制网络 (dlPFC, IPL) | 任务诱发高频活动 |
| 元认知监测 | 前扣带回 (ACC) | 误差相关电位 |
| 元认知控制 | 背外侧前额叶 (dlPFC) | 策略切换信号 |

### 6.2 行为预测

**预测 1**: 元认知训练应改善自我预测误差校准
- 干预: 元认知监测练习
- 预期: 降低社交焦虑中的自我预测误差

**预测 2**: 前反思/反思切换应受认知负荷调节
- 高认知负荷 → 更多依赖前反思
- 低认知负荷 → 更容易进入反思

**预测 3**: 自我预测误差应与主观不适感正相关
- 测量: 自我预测误差 (行为任务)
- 相关: 焦虑/抑郁量表分数

## 七、临床应用方向

### 7.1 焦虑障碍

**假设**: 焦虑障碍 = 慢性高自我预测误差状态

**干预策略**:
1. 元认知训练: 提高对自我预测的置信度校准
2. 暴露疗法: 修正过度警惕的自我预测先验
3. 正念练习: 降低对自我预测误差的过度反应

### 7.2 抑郁症

**假设**: 抑郁症 = 负性自我预测先验过度加权

**干预策略**:
1. 认知重建: 修正负性自我模型
2. 行为激活: 提供反证更新自我预测
3. 元认知疗法: 改变对负性自我想法的关系

### 7.3 人格解体障碍

**假设**: 人格解体 = 自我预测信号传递失败

**干预策略**:
1. 接地技术: 增强内感受预测信号
2. 自我叙事重建: 修复自传体自我预测
3. 身体导向疗法: 强化身体自我预测
