**功能**: 强制自省程序

**四层自省检查**:

| 层 | 检查内容 | 问题示例 |
|---|---------|---------|
| 主动性 | 主动 vs 被动 | "我此刻的行为是主动的还是被动的？" |

| 逻辑思维 | 推理完整性 | "我的推理链条是否完整？(A→B→C)" |
| 升级 | 进步与重复 | "我比上一次输出进步了吗？" |

**核心功能**:
```javascript
reflect(action, context)
// 返回：{ questions, answers, needsImprovement, improvementPlan }

// 返回：格式化的自省报告

// 记录到 data/learning-log.md

// 记录到 data/upgrade-history.md
```

### 2. 自动生成改进计划

**如果检测到需要改进**:
```javascript
if (reflection.needsImprovement) {
  reflection.improvementPlan = createImprovementPlan(answers);
  // 自动执行改进计划
  executeImprovementPlan(reflection.improvementPlan);
}
```

**改进计划示例**:
1. 主动分析用户需求，不等指令

3. 完善推理链条，追溯第一性原理
4. 确保比上一次进步，不重复犯错
