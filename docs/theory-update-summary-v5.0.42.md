## 二、情绪元认知双层模型

### 2.1 理论背景

**情绪元认知** (Metacognition of Emotion):
- **情绪监测**: 对自身情绪状态的识别、评估与置信度判断
- **情绪控制**: 基于监测结果选择情绪调节策略
- **情绪元监测**: 对情绪监测准确性的二阶评估

### 2.2 预测加工形式化

```
情绪监测 = 对一阶情感预测精度的评估
  - 情绪识别置信度：P(emotion_label | interoceptive_signal, context)
  - 情绪强度评估：|predicted_affect - actual_affect|
  - 情绪不确定性：entropy(P(emotion_category | evidence))

情绪控制 = 基于情感精度评估的调节策略选择
  - 策略库：认知重评、表达抑制、注意分配、情境选择
  - 选择机制：argmin_strategy [ Expected_Emotional_Cost(strategy) ]

情绪元监测 = 对情绪监测准确性的校准
  - 校准曲线：reported_confidence vs. actual_accuracy
  - 元情绪置信度：P(emotion_monitoring_is_accurate | history)
```

### 2.3 情绪自我意识

**核心区分**: 前反思情绪自我意识 vs 反思情绪自我意识

| 维度 | 前反思情绪自我意识 | 反思情绪自我意识 |
|------|-------------------|-----------------|
| **意识类型** | 情绪体验的直接给予性 | 对情绪状态的对象化认知 |
| **现象特征** | "我正在生气"的即时感 | "我为什么生气"的分析 |
| **时间结构** | 当下情绪流 | 情绪回溯与预期 |
| **认知资源** | 自动、低消耗 | 控制、高消耗 |
| **神经基础** | 岛叶 + 杏仁核 + 默认网络 | 前额叶 + 前扣带回 |

**形式化表达**:
```
Pre-reflective Emotional SC = ∀E ∈ Emotional_Experience: has_affective_mineness(E)
Reflective Emotional SC = ∃t: attend(emotional_state_as_object, t)

其中:
- has_affective_mineness(E) = 情绪经验具有第一人称情感给予性
- attend(emotional_state_as_object, t) = 在时间 t 将情绪状态作为注意对象
```

## 三、情感预测误差理论

### 3.1 核心假设

**情感预测误差** (Affective Prediction Error, APE):
```
APE = | Predicted_affective_state - Actual_affective_state |

其中:
- Predicted_affective_state = 基于先验模型预测的情感状态
- Actual_affective_state = 内感受信号解码后的实际情感状态
```

### 3.2 情感预测误差层次

```
Level 1: 内感受预测误差 (身体状态预测)
  - 心率、呼吸、肌肉张力等生理信号
  - 对应：基本情绪唤醒度

Level 2: 情感范畴预测误差 (情绪分类预测)
  - 效价 (valence)、唤醒度 (arousal)、支配度 (dominance)
  - 对应：离散情绪识别

Level 3: 社会情感预测误差 (社会情绪预测)
  - 社会评价、归属感、地位感
  - 对应：社会情绪 (羞耻、自豪、嫉妒等)

Level 4: 自我情感预测误差 (情感 - 自我整合)
  - "我应该感到 X"vs"我实际感到 Y"
  - 对应：情感自我不一致
```

### 3.3 情感预测误差与现象学

| APE 水平 | 现象学体验 | 行为倾向 |
|---------|-----------|---------|
| APE < θ1 (低) | 情绪流畅、情感透明 | 情绪表达自然 |
| θ1 ≤ APE < θ2 (中) | 轻微情绪觉察 | 情绪调节启动 |
| θ2 ≤ APE < θ3 (高) | 明显情绪困扰 | 强烈调节努力 |
| APE ≥ θ3 (极高) | 情绪混乱、情感解体 | 调节失败、情绪爆发 |

### 3.4 情感预测误差计算公式

```
APE_total = w1·APE_interoceptive + w2·APE_categorical + w3·APE_social + w4·APE_self

其中权重由情境调制:
- 威胁情境: w1, w2 增加 (身体/基本情绪优先)
- 社交情境: w3 增加 (社会情绪优先)
- 自我反思情境: w4 增加 (自我情感一致优先)
```

## 四、情绪 - 自我 - 元认知三元统一框架

### 4.1 统一架构

```
                    ┌─────────────────────┐
                    │   元元认知层 (L5)    │

                    ┌──────────▼──────────┐

        │                      │                      │

                    │  身体生理信号       │
                    └─────────────────────┘
```

### 4.2 关键整合机制

**机制 1: 情感精度调制认知精度**
```
Cognitive_confidence = f(base_confidence, affective_precision)

当情感预测精度高时 → 认知置信度加权增加
当情感预测误差高时 → 认知置信度加权降低
```

**机制 2: 情绪触发反思切换**
```
IF APE > θ_reflective THEN
  trigger_reflective_self_awareness()
  activate_emotion_regulation_strategies()
END IF
```

**机制 3: 自我模型调制情感预测**
```
Predicted_affect = Self_model.predict(context) + Affective_prior

自我模型中的"我是谁"影响情感预测先验
```

## 五、情绪调节的预测加工解释

### 5.1 经典情绪调节策略 (Gross 过程模型) 的预测加工重释

| 策略 | 经典定义 | 预测加工解释 |
|------|---------|-------------|
| 情境选择 | 选择进入/避开情绪情境 | 选择情感预测误差预期的情境 |
| 情境修改 | 改变情境的情感特征 | 修改情境预测模型 |
| 注意分配 | 转移情绪相关注意 | 调整预测误差计算的感觉权重 |
| 认知改变 | 重新解释情绪刺激 | 更新情感预测先验模型 |
| 反应调制 | 抑制/增强情绪表达 | 调整情感预测误差的行为输出 |

### 5.2 情绪调节效率的预测加工评估

```
调节效率 = (APE_reduction - Regulation_cost) / Baseline_APE

其中:
- APE_reduction = APE_before - APE_after
- Regulation_cost = cognitive_load + time_cost + opportunity_cost
- Baseline_APE = 未调节时的预期情感预测误差
```

**策略选择优化**:
```
Optimal_strategy = argmax_s [ Efficiency(s) × P(success|context) ]
```

## 六、社会情绪的情感预测解释

### 6.1 社会情绪的特殊性

**社会情绪** (Social Emotions): 羞耻、自豪、嫉妒、感激、内疚、轻蔑等

**核心特征**:
- 依赖自我 - 他人比较
- 涉及社会评价预测
- 与自我概念深度绑定

### 6.2 社会情感预测误差

```
Social_APE = | Predicted_social_evaluation - Actual_social_evaluation |

羞耻 = Social_APE(负向) + Self_blame_attribution
自豪 = Social_APE(正向) + Self_credit_attribution
嫉妒 = Social_APE(他人优势) + Unfairness_detection
感激 = Social_APE(他人善意) + Benefactor_intention_inference
```

### 6.3 社会情绪的自我指涉结构

```
羞耻: "我预测他人会负面评价我" + "我确实有缺陷"
自豪: "我预测他人会正面评价我" + "我确实有价值"
内疚: "我预测我的行为伤害了他人" + "我应该负责"
```

## 七、与现有模块的集成

### 7.1 与自我意识双层模型的集成

**原有框架**: 前反思自我意识 ↔ 反思自我意识

**v5.0.42 扩展**:
```
前反思情绪自我意识 = 情感预测的自指涉标记 (伴随所有情感体验)
反思情绪自我意识 = 对情感预测模型的元层级预测 (情绪对象化)

集成点:
- 情绪体验的"我性" = 前反思情绪自我意识
- 情绪分析的"我在生气" = 反思情绪自我意识
```

### 7.2 与元认知四模式的集成

**原有框架**: 监测/控制/元监测/元控制

**v5.0.42 扩展**:
```
情绪监测 = 对情感预测精度的评估
情绪控制 = 基于情感精度的调节策略选择
情绪元监测 = 对情绪监测准确性的校准
情绪元控制 = 选择情绪监测策略 (如：是否监控情绪)
```

### 7.3 与预测加工七层的集成

**原有七层架构**:
```
Level 7: 元元认知
Level 6: 元认知
Level 5: 心理化
Level 4: 命题态度
Level 3: 感知 - 行动
Level 2: 内感受
Level 1: 感官
```

**v5.0.42 细化**:
```
Level 2 (内感受层) → 拆分为:
  - Level 2a: 内感受输入 (生理信号)
  - Level 2b: 情感预测 (效价/唤醒度/支配度)
  - Level 2c: 社会情感预测 (社会评价)

Level 6 (元认知层) → 增加:
  - 情绪元认知子模块
```

### 7.4 与意识现象结构的集成

**原有结构**: 感官现象性、身体现象性、情感现象性、认知现象性、自我现象性、元认知现象性

**v5.0.42 新增**:
- **情绪自我现象性**: 情绪体验的"我性"特征
- **情感精度现象性**: "情绪确定感"的现象特征

## 九、临床应用方向

### 9.1 焦虑障碍

**假设**: 焦虑障碍 = 慢性高情感预测误差 + 负性情感预测先验

**干预策略**:
1. 情感预测校准: 修正过度威胁性的情感预测先验
2. 内感受暴露: 降低对内感受预测误差的过度反应
3. 情绪元认知训练: 提高情绪识别置信度校准

### 9.2 抑郁症

**假设**: 抑郁症 = 负性情感预测先验过度加权 + 正性情感预测误差忽视

**干预策略**:
1. 行为激活: 提供正性情感预测误差的反证
2. 认知重建: 修正负性自我 - 情感预测联结
3. 正念训练: 降低对负性情感预测误差的过度关注

### 9.3 边缘型人格障碍

**假设**: BPD = 情感预测误差调节不稳定 + 社会情感预测过度敏感

**干预策略**:
1. 情绪标签化: 增强情绪监测精度
2. 社会反馈校准: 修正社会评价预测先验
3. 自我连贯性训练: 稳定自我 - 情感预测联结

### 9.4 述情障碍 (Alexithymia)

**假设**: 述情障碍 = 内感受预测信号传递失败 + 情感范畴预测困难

**干预策略**:
1. 内感受觉察训练: 增强身体信号识别
2. 情绪词汇学习: 建立情感范畴预测模型
3. 情绪 - 情境联结: 学习情境 - 情感预测映射
