## 🎯 升级内容

### 1. 依恋风格识别系统

基于依恋理论 (Bowlby, 1969)，识别用户的依恋风格：

```javascript
// 三种依恋风格
const AttachmentStyles = {
  SECURE: '安全型',      // 信任他人，容易建立亲密关系
  ANXIOUS: '焦虑型',     // 担心被抛弃，需要频繁确认
  AVOIDANT: '回避型'     // 避免亲密，保持情感距离
};
```

**识别方式**：
- 分析用户对话中的关系模式
- 检测关键词和情感表达
- 动态调整回应方式

### 2. 马斯洛需求层次检测

识别用户当前需求层次：

```
层次 5: 自我实现 → "我想成为更好的自己"
层次 4: 尊重需求 → "我希望被认可"
层次 3: 归属需求 → "我想要被接纳"
层次 2: 安全需求 → "我需要稳定"
层次 1: 生理需求 → "我累了/饿了/困了"
```

### 3. 优化信任度计算

```javascript
// v2.2.0 新增权重
- 用户表达脆弱 → 信任度 +3
- 用户追问深层问题 → 信任度 +2
- 用户表达感谢 → 信任度 +1
- 用户主动分享 → 信任度 +2
```

### 4. 增加回应模板

#### 安全型依恋回应
```
"你的感受很重要，我在这里听你说。"
"你愿意分享这些，说明你在建立信任，这很珍贵。"
```

#### 焦虑型依恋支持
```
"我注意到你很在意这段关系，这是正常的。"
"你的担心是可以理解的，同时也有其他可能性。"
```

#### 回避型依恋尊重
```
"我理解你需要一些空间，这完全没问题。"
"你可以按照自己的节奏来，我在这里但不打扰。"
```
