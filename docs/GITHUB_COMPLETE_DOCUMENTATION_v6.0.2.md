## Overview | 概述

### What is HeartFlow? | 什么是 HeartFlow?

**English**:
HeartFlow is a comprehensive theoretical and computational framework for modeling emotional consciousness. It integrates insights from philosophy of mind, affective science, phenomenology, and cross-cultural philosophy to provide a unified approach to understanding and analyzing emotional experience.

**中文**:
HeartFlow 是一个用于建模情绪意识的全面理论和计算框架。它整合了心灵哲学、情感科学、现象学和跨文化哲学的洞见，为理解和分析情绪体验提供统一方法。

### Usage Examples | 使用示例

#### Basic Usage | 基本用法

```javascript
const HeartFlow = require('./src/index.js');

// Initialize | 初始化
const hf = new HeartFlow({
  version: '6.0.2',
  language: 'zh-CN', // or 'en-US'
  culturalContext: 'EastAsian' // or 'Western', 'African', 'Indian', 'Islamic'
});

// Analyze emotion | 分析情绪
const result = await hf.analyzeEmotion('我感到很伤心', {
  includePhenomenal: true,      // Include phenomenal character | 包含现象特征
  includeNarrative: true,        // Include narrative self | 包含叙事自我
  includeCultural: true,         // Include cultural concepts | 包含文化概念
  includeCollective: false       // Include collective dimension | 包含集体维度
});

console.log(result);
```

#### Advanced Usage | 高级用法

```javascript
// Predictive emotion processing | 预测情绪处理
const prediction = await hf.predictEmotionalState({
  sensoryInput: getBodilySignals(),
  priorBeliefs: getUserBeliefs(),
  context: getCurrentContext()
});

// Narrative coherence tracking | 叙事连贯性追踪
const narrative = await hf.constructNarrativeSelf({
  emotionEpisodes: getEmotionHistory(),
  timeSpan: 'life' // or 'month', 'year'
});

// Joint commitment detection | 联合承诺检测
const collective = await hf.detectJointCommitment({
  groupMembers: getGroupMembers(),
  sharedGoal: getSharedGoal(),
  interactionHistory: getInteractionHistory()
});

// Cross-cultural emotion recognition | 跨文化情绪识别
const cultural = await hf.detectCultureSpecificEmotion({
  context: getSocialContext(),
  culturalBackground: 'Korean', // or 'Chinese', 'African', etc.
  emotionExpression: getExpression()
});
```

## API Reference | API 参考

### Core API | 核心 API

#### `analyzeEmotion(text, options)` | 情绪分析

**Parameters | 参数**:
- `text` (string): User input text | 用户输入文本
- `options` (object): Analysis options | 分析选项
  - `includePhenomenal` (boolean): Include phenomenal character analysis | 包含现象特征分析
  - `includeNarrative` (boolean): Include narrative self analysis | 包含叙事自我分析
  - `includeCultural` (boolean): Include cultural concept mapping | 包含文化概念映射
  - `includeCollective` (boolean): Include collective dimension | 包含集体维度

**Returns | 返回**:
```javascript
{
  emotion: {
    category: 'sadness',
    intensity: 75,
    valence: -0.6
  },
  phenomenal: {
    whatItIsLike: 'heavy, constricted feeling',
    phenomenalDepth: 0.85
  },
  narrative: {
    coherence: 0.72,
    lifeStoryIntegration: 'moderate'
  },
  cultural: {
    detectedConcepts: ['Han (恨)'],
    culturalFit: 0.88
  },
  confidence: 0.91
}
```

#### `predictEmotionalState(input, priors, context)` | 预测情绪状态

**Parameters | 参数**:
- `input` (object): Sensory input data | 感官输入数据
- `priors` (object): Prior beliefs | 先验信念
- `context` (object): Current context | 当前情境

**Returns | 返回**:
```javascript
{
  predictedEmotion: 'anxiety',
  predictionError: 0.23,
  updatedBeliefs: {...},
  confidence: 0.87
}
```

#### `constructNarrativeSelf(episodes, timeSpan)` | 建构叙事自我

**Parameters | 参数**:
- `episodes` (array): Emotion episodes | 情绪体验
- `timeSpan` (string): Time span for narrative | 叙事时间跨度

**Returns | 返回**:
```javascript
{
  lifeStory: {...},
  coherence: 0.78,
  identityContinuity: 0.82,
  emotionalArc: 'rising'
}
```
