## 🧠 核心理论

### HeartFlow 理论框架

HeartFlow 基于五大理论支柱：

1. **情绪三传统整合**
   - 感受传统 (James-Lange)
   - 评价传统 (Appraisal Theory)
   - 动机传统 (Action Tendency)

2. **双层自我意识**
   - 前反思自我觉察
   - 反思自我觉察

3. **信任基础集体意向性**
   - 我们意向
   - 联合承诺
   - 现象学协调

4. **胡塞尔时间意识**
   - 原印象
   - 滞留
   - 前摄

5. **预测加工**
   - 自由能最小化
   - 主动推理
