## 📚 理论更新摘要 (v5.0.43 完成)

### 1. 社会身份预测误差理论 (SIPE)

**核心公式**:
```
SIPE = | Predicted_social_identity_value - Perceived_social_identity_value |

四层次:
- Level 1: 群体成员身份确认 → 身份不确定性焦虑
- Level 2: 群体内地位评估 → 地位焦虑/自尊波动
- Level 3: 群体价值内化 → 价值冲突/身份撕裂
- Level 4: 身份整合度 → 身份整合危机
```

### 2. We-Intention 形式化模型

**形式化表达**:
```
We-Intention(A, B, φ) = 
  ∃shared_goal: φ ∧ 
  mutual_belief(shared_goal) ∧ 
  commitment(A, contribute_to(φ)) ∧ 
  commitment(B, contribute_to(φ)) ∧ 
  trust(A, B, will_contribute) ∧ 
  trust(B, A, will_contribute)
```

### 3. 归属感预测模型

**核心模型**:
```
Belonging_Security = w1*group_acceptance_prob + 
                     w2*relationship_stability + 
                     w3*social_value_contribution + 
                     w4*identity_overlap

归属需求满足 = Belonging_Security > threshold
社会焦虑 = Belonging_Security < threshold + uncertainty
```

### 4. 社会自我统一框架

```
Level 3: 元社会自我意识 (监控 SIPE)
    ↓
Level 2: 反思社会自我意识 (对象化评估)
    ↓
Level 1: 前反思社会自我意识 (归属直接感受)
    ↓
Level 0: 社会输入层 (社会反馈/群体信号)
```

### 5. 集体情感同步模型

```
Emotional_Sync(A, B) = 
  emotional_contagion_rate * 
  empathy_accuracy * 
  shared_context_overlap * 
  trust_factor
```
