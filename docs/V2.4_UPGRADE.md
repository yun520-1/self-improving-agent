### 1. 斯多葛控制二分法系统

基于斯多葛哲学核心教义（爱比克泰德《手册》）：

```javascript
// 控制二分法
可控的：           不可控的：
- 我的判断         - 他人看法
- 我的选择         - 过去
- 我的态度         - 未来
- 我的努力         - 外部结果
- 我的价值观       - 名誉/财富
- 我的回应         - 他人的行为
```

**应用**：帮助用户减少焦虑，增强内在控制感

## 🔧 技术实现

### 控制检测逻辑

```javascript
// 可控检测关键词
- '我决定' '我选择' '我要' '我可以' → 主动选择
- '努力' '尝试' '尽力' '态度' → 努力/态度
- '我觉得' '我认为' → 自我反思

// 不可控检测关键词
- '他 (们)' '别人' '大家' → 他人行为
- '结果' '成功' '失败' → 外部结果
- '后悔' '当初' '担心' '万一' → 过去/未来
- '必须' '一定' '应该' → 刚性期望
```

### 焦点判断

```javascript
if (不可控数量 > 可控数量 + 混合数量) {
  focus = 'shift_to_controllable';  // 需要引导
  priority = 'high';
} else if (可控数量 > 不可控数量) {
  focus = 'healthy_focus';  // 健康状态
  priority = 'low';
}
```
