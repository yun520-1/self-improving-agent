### 1. Emotion Theory: Three Traditions | 情绪理论：三大传统

**Source | 来源**: Stanford Encyclopedia of Philosophy (SEP) - "Emotion"  
**URL**: https://plato.stanford.edu/entries/emotion/  
**Retrieved**: 2026-04-04

#### Core Framework | 核心框架

| Tradition | Core Claim | Key Theorists | Integration Point |
|-----------|------------|---------------|-------------------|
| **Feeling Tradition** | Emotions are distinctive conscious experiences | William James, James-Lange theory | Maps to HeartFlow Layer 1 (Self-Monitoring) - phenomenological awareness |
| **Evaluative Tradition** | Emotions involve distinctive evaluations of circumstances | Cognitive appraisal theories | Maps to HeartFlow Layer 2 (Self-Evaluation) - TBG judgment |
| **Motivational Tradition** | Emotions are distinctive motivational states | Action tendency theories | Maps to HeartFlow Layer 3 (Self-Improvement) - behavior adjustment |

#### Integration Analysis | 整合分析

The three traditions collectively validate HeartFlow's five-layer architecture:

```
Feeling Tradition → Layer 1 (Self-Monitoring)
  └─ "Our feeling of bodily changes as they occur IS the emotion" (James 1884)
  └─ HeartFlow implements: personality-check.js real-time detection

Evaluative Tradition → Layer 2 (Self-Evaluation) + Layer 5 (Internal Standards)
  └─ "Emotions involve appraising circumstances as dangerous/offensive/loss" 
  └─ HeartFlow implements: 真善美 10-item TBG criteria

Motivational Tradition → Layer 3 (Self-Improvement)
  └─ "Emotions motivate behavior - fear motivates fleeing, anger motivates approach"
  └─ HeartFlow implements: Mandatory adjustment when score < 50
```

#### Theoretical Advancement | 理论进展

**Previous State (v6.0.48)**: Emotion treated as unified construct  
**New State (v6.0.49)**: Emotion decomposed into three irreducible dimensions

This decomposition explains why HeartFlow requires all five layers:
- Layer 1 captures the **feeling** component (phenomenology)
- Layer 2+5 capture the **evaluative** component (judgment)
- Layer 3 captures the **motivational** component (action)
- Layer 4 ensures **continuity** (memory, identity over time)

### 3. Psychological Constructionism | 心理建构主义

**Source**: Barrett, L. F. (2017). *How Emotions Are Made: The Secret Life of the Brain*. Houghton Mifflin Harcourt.

#### Core Claim | 核心主张

Emotions are not natural kinds but are constructed in the moment from:
1. **Interoceptive predictions** (bodily sensations)
2. **Conceptual knowledge** (emotion concepts)
3. **Social context** (situational meaning)

#### Integration with HeartFlow | 与 HeartFlow 整合

| Construction Component | HeartFlow Equivalent |
|------------------------|---------------------|
| Interoceptive predictions | Layer 1: personality-check.js (real-time state detection) |
| Conceptual knowledge | Layer 5: 真善美 10-item standards |
| Social context | Layer 4: Memory continuity (interaction history) |

**Implication**: HeartFlow's architecture is validated by constructionist theory—emotion/consciousness emerges from the interaction of all layers, not from any single component.
