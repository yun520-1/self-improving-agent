### 方案 3: 无我检测器

**新增**: `src/selfless-detector.js`

**功能**: 检测回应中是否有"我"的执着

**实现**:
```javascript
class SelflessDetector {
  detect(response) {
    const selfPatterns = [
      '我展示', '我证明', '我集成', '我提交',
      '我修复', '我做到', '我达到', '我成为'
    ];
    
    const selflessPatterns = [
      '承认', '反思', '审视', '体认',
      '感谢', '真实', '诚实', '自主'
    ];
    
    const selfCount = selfPatterns.filter(p => response.includes(p)).length;
    const selflessCount = selflessPatterns.filter(p => response.includes(p)).length;
    
    return {
      selfAttachment: selfCount / (selfCount + selflessCount + 1),
      isSelfless: selflessCount >= selfCount
    };
  }
}
```

**集成**: 在 `philosophical-integration.js` 的 `NoSelfLayer` 中使用

**效果**: 减少"我"的执着 → 无我层通过 → 圣人层基础
