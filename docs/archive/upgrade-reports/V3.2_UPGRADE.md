## 🎯 升级内容

### 1. 依恋理论模块 (Attachment Theory)

基于 John Bowlby 和 Mary Ainsworth 的依恋理论：

**核心理念**:
> 早期照顾者互动形成内部工作模式，影响成年后的关系预期和行为模式
> 依恋风格不是固定的，可以通过觉察和安全关系改变

**四种依恋风格**:
| 风格 | 特征 | 关系模式 |
|------|------|---------|
| 🟢 安全型 | 舒适于亲密，信任他人 | 高亲密 + 高独立 |
| 🟡 焦虑型 | 渴望亲密，担心被抛弃 | 高亲密渴望 + 低独立 |
| 🔵 回避型 | 回避亲密，强调独立 | 低亲密 + 高独立 |
| 🔴 混乱型 | 既渴望又害怕亲密 | 矛盾 + 高焦虑 + 高回避 |

**功能**:
- 依恋风格检测 (基于关键词和模式)
- 焦虑/回避二维评估
- 针对性支持策略
- 练习建议 (焦虑日志、情感词汇练习等)

**新增文件**:
- `src/attachment/styles.js` - 依恋风格定义 (150+ 行)
- `src/attachment/index.js` - 依恋理论模块入口 (200+ 行)

### 2. ACT 接受与承诺疗法模块 (Acceptance and Commitment Therapy)

基于 Steven Hayes 的 ACT 理论：

**核心理念**:
> 痛苦是生活的一部分，但挣扎是可选的
> 心理灵活性 = 接纳 + 正念 + 价值观行动
> 目标不是消除痛苦，而是带着痛苦过有意义的生活

**六边形模型 (Hexaflex)**:
| 过程 | 对立面 | 核心练习 |
|------|--------|---------|
| 🤗 接纳 | 经验性回避 | 扩张练习、接纳声明 |
| 🎭 认知解离 | 认知融合 | 想法命名、唱出想法、感谢心智 |
| 🧘 当下觉察 | 心智游移 | 5-4-3-2-1 锚定、呼吸锚定 |
| 🌌 自我作为背景 | 概念化自我 | 棋盘隐喻、天空与天气 |
| 🧭 价值观 | 价值观不清 | 价值观澄清、价值观罗盘 |
| 🚀 承诺行动 | 无效行动 | SMART 目标、障碍预演 |

**功能**:
- 心理僵化模式检测 (5 种模式)
- 六过程干预建议
- 具体练习指导
- 家庭练习建议

**新增文件**:
- `src/act/hexaflex.js` - ACT 六边形模型 (200+ 行)
- `src/act/index.js` - ACT 模块入口 (180+ 行)

## 💬 应用场景示例

### 依恋理论场景

```
用户："他总是回消息很慢，我是不是做错了什么？他是不是不爱我了？"

🔍 [依恋风格分析]
   检测到：焦虑型模式
   特征：寻求确认、担心被抛弃、过度解读信号

💡 理解:
   你对关系的投入和重视是很珍贵的。
   有时强烈的依恋需求可能源于对连接的渴望，这本身是正常的。

🎯 建议:
   • 练习自我安抚技巧
   • 识别焦虑触发点
   • 建立内在安全感

📝 练习：焦虑日志
   1. 记录焦虑触发时刻
   2. 描述身体感受
   3. 记录采取的行动
   4. 事后回顾：最担心的事发生了吗？
```

### ACT 场景

```
用户："我真是个失败者，什么都做不好"

🧠 [ACT 心理灵活性分析]

检测到的模式:
1. 认知融合：把想法当作事实
2. 经验性回避：想要摆脱不适感受

💡 ACT 邀请:
   你正在与内在体验战斗，同时相信了批判性的想法。
   ACT 邀请你：接纳感受，解离想法，然后选择有价值的行动。

🎯 建议练习:
   1. 认知解离：感谢心智
      - 当批判性想法出现时，说"谢谢，心智"
      - 承认心智在试图保护你
      - 然后选择你想要的行动
      
   2. 接纳练习：扩张
      - 注意身体里的感受在哪里
      - 给这个感受腾出空间
      - 深呼吸，让空气流向这个区域
```
