## 📝 理论更新摘要

### 1. 自我意识双层模型精细化

**核心区分**: 前反思自我意识 vs 反思自我意识

| 维度 | 前反思自我意识 | 反思自我意识 |
|------|---------------|-------------|
| **意识类型** | 非对象化、伴随性 | 对象化、主题化 |
| **时间结构** | 当下即时 | 延时回溯 |
| **认知资源** | 自动、低消耗 | 控制、高消耗 |
| **现象特征** | "我性"(mineness) | "自我模型"(self-model) |
| **神经基础** | 默认网络 + 突显网络 | 额顶控制网络 |

**形式化表达**:
```
Pre-reflective SC = ∀E ∈ Experience: has_mineness(E)
Reflective SC = ∃t: attend(self_as_object, t)

其中:
- has_mineness(E) = 经验 E 具有第一人称给予性
- attend(self_as_object, t) = 在时间 t 将自我作为注意对象
```

**与预测加工的整合**:
- 前反思自我意识 = 预测模型的隐性自我指涉结构
- 反思自我意识 = 对预测模型本身的元层级预测

### 2. 元认知四模式计算形式化

**四模式架构**:

```
Level 4: 元元认知 (关于元认知的认知)
    ↓
Level 3: 策略选择 (选择认知策略)
    ↓
Level 2: 监测评估 (评估认知状态)
    ↓
Level 1: 对象认知 (目标认知任务)
```

**元认知监测公式**:
```
Meta-Monitoring = f(Confidence, Difficulty, Progress, Uncertainty)

Confidence = P(correct | evidence, prior)
Difficulty = 1 - P(success | current_ability, task_demand)
Progress = (current_state - initial_state) / (goal_state - initial_state)
Uncertainty = entropy(P(outcome | current_model))
```

**元认知控制公式**:
```
Meta-Control = argmax_strategy [ Expected_Utility(strategy) - Cost(strategy) ]

Expected_Utility = P(success | strategy) × Value(success)
Cost = cognitive_load + time_cost + opportunity_cost
```

**四模式应用**:
| 模式 | 功能 | 计算实现 |
|------|------|---------|
| 监测 | 评估当前认知状态 | 置信度计算、误差检测 |
| 控制 | 调整认知策略 | 资源分配、策略切换 |
| 元监测 | 评估监测的准确性 | 校准曲线、Brier 分数 |
| 元控制 | 选择监测策略 | 监测频率优化、阈值调整 |

### 3. 自我意识 - 元认知统一框架

**统一架构**:
```
                    ┌─────────────────────┐
                    │   元元认知层 (L4)    │
                    │  关于元认知的认知    │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │    元认知层 (L3)     │
                    │  监测 + 控制循环     │
                    └──────────┬──────────┘
                               │
        ┌──────────────────────┼──────────────────────┐
        │                      │                      │
┌───────▼───────┐    ┌────────▼────────┐    ┌───────▼───────┐
│  反思自我意识  │    │   对象认知层    │    │  前反思自我意识 │
│   (对象化)    │    │   (目标任务)    │    │   (伴随性)    │
└───────────────┘    └─────────────────┘    └───────────────┘
```

**关键整合点**:
- 元认知监测依赖前反思自我意识提供的"我性"信号
- 反思自我意识是元认知控制的对象化产物
- 元元认知是对自我意识状态的二阶表征

### 4. 自我意识的预测加工解释

**核心假设**: 自我意识是预测模型对自身结构的自指涉建模

**自指涉预测层次**:
```
Level 0: 世界模型预测 (外部刺激)
Level 1: 身体模型预测 (本体感觉、内感受)
Level 2: 行动模型预测 (运动命令、预期感觉)
Level 3: 心理模型预测 (信念、欲望、意图)
Level 4: 自我模型预测 (关于"我"的预测)
Level 5: 元自我模型预测 (关于自我模型的预测)
```

**自我预测误差**:
```
Self-PE = | Predicted_self_state - Actual_self_state |

当 Self-PE 超过阈值时 → 触发反思自我意识
当 Self-PE 持续低位时 → 维持前反思自我意识
```

**现象学对应**:
| 预测状态 | 现象学体验 |
|---------|-----------|
| 低自我预测误差 | 流畅的自我感、行动透明性 |
| 中等自我预测误差 | 轻微自我意识、可被忽略 |
| 高自我预测误差 | 强烈自我意识、行动不透明 |
| 预测失败 | 自我感丧失、人格解体 |

```javascript
// 自我预测误差计算
function calculateSelfPredictionError(currentState, selfModel) {
  const predictedSelf = selfModel.predict(currentState.context);
  const actualSelf = currentState.selfRepresentation;
  
  const selfPE = divergence(predictedSelf, actualSelf);
  
  // 根据 PE 大小决定意识类型
  if (selfPE > REFLECTIVE_THRESHOLD) {
    return { type: 'reflective', intensity: selfPE };
  } else {
    return { type: 'pre-reflective', intensity: selfPE };
  }
}

// 元认知监测
function metaCognitiveMonitoring(taskState, confidence) {
  return {
    confidence: calculateConfidence(taskState.evidence),
    difficulty: estimateDifficulty(taskState, agentAbility),
    progress: calculateProgress(taskState.current, taskState.goal),
    uncertainty: calculateEntropy(taskState.outcomeDistribution)
  };
}

// 元认知控制
function metaCognitiveControl(monitoringResult, strategies) {
  const utilities = strategies.map(s => ({
    strategy: s,
    utility: expectedUtility(s, monitoringResult) - cost(s)
  }));
  
  return argmax(utilities, u => u.utility);
}
```
