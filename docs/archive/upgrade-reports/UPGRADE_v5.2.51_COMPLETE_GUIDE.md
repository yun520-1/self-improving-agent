### 2. 时间意识：现象学结构整合 | Temporal Consciousness: Phenomenological Structure

#### 胡塞尔时间三重结构 | Husserlian Tripartite Time Structure

```
时间意识结构 | Temporal Consciousness Structure:

        过去              现在              未来
        Past            Present          Future
         ↓                ↓                 ↓
      保留            原印象            前摄
   (Retention)   (Primal Impression)  (Protention)
```

**核心概念 | Core Concepts**:
- **保留 | Retention**: 对刚刚过去的即刻意识（如音符的余响）
- **原印象 | Primal Impression**: 当前的直接体验
- **前摄 | Protention**: 对即将到来的预期（如期待下一个音符）

**显似现在 | Specious Present**:
- **持续时间 | Duration**: ~2-3 秒（情绪体验窗口）
- **应用 | Application**: 情绪追踪的时间粒度

#### 时间 - 情绪整合 | Temporal-Emotion Integration

```
时间导向情绪分类 | Temporal Orientation of Emotions:

过去导向情绪 | Past-Oriented:
• 后悔 (Regret) - 对过去行为的负面评价
• 悔恨 (Remorse) - 道德性过去导向情绪
• 羞耻 (Shame) - 自我相关的过去评价
• 怀旧 (Nostalgia) - 苦乐参半的过去怀念

现在导向情绪 | Present-Oriented:
• 喜悦 (Joy) - 当前积极体验
• 恐惧 (Fear) - 当前威胁响应
• 愤怒 (Anger) - 当前冒犯响应

未来导向情绪 | Future-Oriented:
• 希望 (Hope) - 积极未来预期
• 恐惧/ dread (Dread) - 消极未来预期

• 期待 (Anticipation) - 未来事件准备
```

### 5. 集体意向性：我们意向系统 | Collective Intentionality: We-Intention System

#### 我们意向架构 | We-Intention Architecture

**核心洞见 | Core Insight**:
集体意向**不可还原**为个体意向的简单加总。

```
错误理解 | Incorrect:
"我们意图 J" = "我意图 J" + "你意图 J" + "我们知道这一点"
"We intend to J" = "I intend to J" + "You intend to J" + "We know this"

正确理解 | Correct:
"我们意图 J" = 不可还原的集体意向状态
"We intend to J" = Irreducible collective intentional state
```

#### 三种理论账户 | Three Theoretical Accounts

**1. 内容账户（布兰特曼）| Content Account (Bratman)**:
```
共享意向结构 | Shared Intention Structure:

"我意图我们 J"
"I intend that we J"
    +
相互响应 (Mutual responsiveness)
    +
协调子计划 (Meshing subplans)
    +
共同知识 (Common knowledge)
```

**2. 模式账户（图梅拉/塞尔）| Mode Account (Tuomela/Searle)**:
```
我们模式 vs. 我模式 | We-mode vs. I-mode:

我们模式特征 | We-mode Features:
• 集体承诺 | Collective commitment
• 群体认同 | Group identification
• "我们"作为行动主体 | "We" as agent
```

**3. 现象学账户（谢勒/瓦尔特）| Phenomenological Account (Scheler/Walther)**:
```
谢勒：数值相同状态 | Scheler: Numerically Identical States:

父母在孩子病床前的悲伤：
不是两个平行的悲伤体验
而是共享的一个悲伤体验

Parents grieving at child's deathbed:
Not two parallel grief experiences
But ONE shared grief experience
```

**瓦尔特四层共享体验 | Walther's Four-Layer Shared Experience**:
```
Layer 1: A 体验 x，B 体验 x
         A experiences x, B experiences x

Layer 2: A 共情 B 的体验，B 共情 A 的体验
         A empathizes with B's experience, vice versa

Layer 3: A 认同 B 的体验，B 认同 A 的体验
         A identifies with B's experience, vice versa

Layer 4: 对认同的相互共情意识
         Mutual empathetic awareness of identification
```
