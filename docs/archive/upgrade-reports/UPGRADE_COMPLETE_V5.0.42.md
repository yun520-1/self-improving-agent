## 📊 理论更新摘要

### 核心进展

**1. 情绪 - 自我意识整合框架**
- 情绪作为自我预测误差的具身信号
- 情绪粒度与自我意识层次对应 (6 层情绪 - 自我映射)
- 情绪调节 = 自我预测误差的动态校准

**2. 情绪元认知四维度**
- 情绪监测：追踪情绪状态 (强度、效价、唤醒度、身体感觉)
- 情绪评估：评价情绪合理性 (情境匹配、强度适当、功能适应)
- 情绪控制：调节情绪反应 (重评、表达抑制、注意部署、反应调节)
- 情绪元监测：评估情绪觉察准确性 (情绪粒度、分辨力、盲点检测)

**3. 情感预测误差理论**
- Self-PE 情绪版本：情绪体验源于自我预测与实际状态的误差
- 情绪障碍的预测加工解释：焦虑 (高估威胁 PE)、抑郁 (低估奖励 PE)
- 情绪调节的预测模型：更新先验 vs 改变输入 vs 调整精度

**4. 情绪 - 自我六层映射**
```
Layer 0-1: 基础情绪 (愉悦/不悦) ←→ 前反思自我感
Layer 2-3: 社会情绪 (羞耻/自豪) ←→ 社会自我模型
Layer 4-5: 元情绪 (对情绪的情绪) ←→ 反思自我意识
```

**5. 情绪粒度与自我分化**
- 高情绪粒度 = 高自我分化 = 更好心理健康
- 情绪词汇库扩展训练 (60+ 精确情绪词)
- 身体感觉 - 情绪映射精细化

**6. 情绪调节的元认知控制**
- 策略选择基于情境评估
- 灵活性 = 心理健康核心指标
- 元认知监控策略效果
