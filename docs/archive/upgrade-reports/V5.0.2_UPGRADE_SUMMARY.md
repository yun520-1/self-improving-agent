## 🧠 理论精华整合

### 情绪理论 (SEP Emotion)

**三大传统整合**:
```
情绪 = 感受 (Feeling) + 评价 (Evaluative) + 动机 (Motivational)

- Feeling: 主观体验质量、强度、效价
- Evaluative: 情境解释、意义建构、价值判断
- Motivational: 行动倾向、目标相关性、紧迫性
```

**临床应用**:
- 感受主导 → 正念、身体聚焦干预
- 评价主导 → 认知重构、意义治疗
- 动机主导 → 行为激活、暴露疗法

---

### 自我意识现象学 (SEP Self-Consciousness)

**层次结构**:
```
前反思层 (基础) → 反思层 (次级) → 社会层 (高级)

- 前反思：非对象化、第一人称给定性、即时性
- 反思：元认知、自我观察、分析
- 社会：他者视角、关系性自我
```

**干预匹配**:
- 沉浸模式 (前反思高/反思低) → 发展元认知
- 疏离模式 (反思高/前反思低) → reconnect with experience
- 整合模式 (两者平衡) → 深化整合

---

### 集体意向性 (SEP Collective Intentionality)

**关键洞察**:
```
We-Intention ≠ Σ Individual Intentions

- Searle: 原初的"我们意图"
- Gilbert: 联合承诺形成 plural subject
- Bratman: 相互响应的子计划系统
```

**应用方向**:
- 关系咨询中的"我们感"培养
- 团队动力分析
- 社会认同与集体行动
