## 📝 理论更新摘要

### 1. 预测加工意识理论 (Predictive Processing Consciousness)

**核心框架**: 基于 Karl Friston 自由能原理的意识理论

**关键命题**:
1. 意识内容 = 主导性预测模型的现象学表现
2. 注意 = 精度加权的分配机制
3. 自由意志现象学 = 主动推理的主观体验

**与 v5.0.31 四维框架整合**:
- 现象性维度 ← 多层级预测模型的现象学特征
- 取用性维度 ← 全局工作空间的预测误差广播
- 现象结构维度 ← 预测层次结构的时空组织
- 元认知维度 ← 精度加权的二阶监控

### 2. 全局工作空间理论更新 (GWT 2025)

**精度加权广播机制**:
- 意识程度 = 广播范围 × 精度权重
- 前额叶 - 顶叶网络 = 全局工作空间神经相关物
- 伽马波段同步 = 信息广播的神经标记

### 3. 整合信息理论 4.0 (IIT 4.0)

**Φ值的因果结构分析**:
- Φ = 系统因果力的不可还原性度量
- 与现象学四维框架对应 (存在性/成分性/信息性/整合性/排他性)

### 4. 高阶思维理论深化 (HOT 2025)

**内在 vs 外在高阶思维**:
- 内在 HOT ← 前反思自我意识 (非对象化)
- 外在 HOT ← 反思自我意识 (对象化)

### 5. 意识与注意的区分 (2025)

**现象性意识 vs 取用性意识**:
- P 意识可无注意，A 意识需注意
- 预测加工解释注意 - 意识解离现象
