## 🎯 升级内容

### 1. 正念与接纳模块 (Mindfulness & Acceptance)

基于正念减压疗法 (MBSR) 和接纳承诺疗法 (ACT) 的完整实现。

#### 正念七原则

| 原则 | 英文 | 说明 |
|------|------|------|
| 不评判 | Non-judging | 对体验不做评价，只是觉察 |
| 耐心 | Patience | 允许事物按其节奏展开 |
| 初心 | Beginner's Mind | 以新鲜的眼光看待每一刻 |
| 信任 | Trust | 信任自己的直觉和体验 |
| 不强求 | Non-striving | 不试图达到某种状态 |
| 接纳 | Acceptance | 如实地接纳当下 |
| 放下 | Letting Go | 不执着于想法或感受 |

#### 五种核心练习

| 练习 | 时长 | 应用场景 |
|------|------|---------|
| 觉察呼吸 | 3-5 分钟 | 焦虑、紧张、压力 |
| 身体扫描 | 5-10 分钟 | 身心连接、放松 |
| 五感着陆法 | 2-3 分钟 | 急性焦虑、恐慌 |
| 思绪如云 | 3-5 分钟 | 过度思考、思绪纷乱 |
| 慈心冥想 | 5-10 分钟 | 自我批评、自我关怀 |

#### ACT 核心流程

```
接纳 (Accept) → 选择方向 (Choose) → 采取行动 (Take Action)
```

**接纳**: 接纳你的体验，而不是与之抗争  
**选择**: 选择对你重要的价值方向  
**行动**: 按照你的价值采取行动

### 2. 成长型思维 (Growth Mindset)

基于卡罗尔·德韦克的终身成长理论。

#### 思维转换表

| 固定型思维 | 成长型思维 |
|-----------|-----------|
| "我做不到" | "我暂时还做不到" |
| "这太难了" | "这需要更多时间和练习" |
| "我犯了错误" | "错误是学习的机会" |
| "我不擅长这个" | "我可以通过练习变得更好" |
| "这不够好" | "我还能如何改进？" |
| "我放弃了" | "我会尝试不同的方法" |

#### 核心理念

> "成为一个人，不是到达一个终点，而是走向一个方向。" — 卡罗尔·德韦克

## 🔧 技术实现

### 自动检测与响应

```javascript
// 检测焦虑/压力
if (input.includes('焦虑') || input.includes('紧张')) {
  return {
    type: 'anxiety',
    exercise: 'breathing',
    response: generateAnxietyResponse()
  };
}

// 检测过度思考
if (input.includes('想太多') || input.includes('脑子乱')) {
  return {
    type: 'overthinking',
    exercise: 'thoughtsAsClouds',
    response: generateOverthinkingResponse()
  };
}

// 检测自我批评
if (input.includes('我不够好') || input.includes('我真没用')) {
  return {
    type: 'selfCriticism',
    exercise: 'lovingKindness',
    response: generateSelfCompassionResponse()
  };
}

// 检测成长型思维机会
const fixedMindsetPatterns = ['做不到', '太难', '放弃了', '不擅长'];
for (const pattern of fixedMindsetPatterns) {
  if (input.includes(pattern)) {
    return {
      type: 'mindsetShift',
      shift: findMindsetShift(pattern),
      response: generateGrowthMindsetResponse()
    };
  }
}
```

### 回应示例

**焦虑回应**:
```
🧘 [正念练习：觉察呼吸]

我注意到你此刻可能感到焦虑或紧张。让我们做一个简单的呼吸练习：

1. 找一个舒适的姿势
2. 将注意力带到呼吸上
3. 感受气息进出身体的感觉
4. 当思绪飘走时，温和地带回呼吸

💬 "呼吸是锚，思绪是云。云会飘走，锚始终在这里。"
```

**成长型思维回应**:
```
🌱 [成长型思维]

我注意到一个可以转换的思维方式：

❌ 固定型思维："我做不到"
✅ 成长型思维："我暂时还做不到"

💬 卡罗尔·德韦克："成为一个人，不是到达一个终点，
   而是走向一个方向。"
```
