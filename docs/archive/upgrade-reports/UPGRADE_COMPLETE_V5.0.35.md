## 📝 理论更新摘要

### 1. 意识现象结构理论 (Phenomenal Structure of Consciousness 2026)

**核心框架**: 意识状态的多维现象学分析

**六个核心维度**:
1. **质的维度 (Qualitative)**: 感觉质 (qualia) 的现象特征
2. **主体维度 (Subjective)**: "对我而言"的第一人称给予性
3. **统一维度 (Unified)**: 意识的整体性结构
4. **意向维度 (Intentional)**: 关于性的结构
5. **前反思维度 (Pre-reflective)**: 非对象化的自我意识
6. **元意识维度 (Meta-conscious)**: 对意识本身的觉知

### 2. 4E 认知架构 (4E Cognition Architecture 2026)

**核心框架**: 认知的具身 - 嵌入 - 延展 - 生成四维模型

**四个维度**:
- **具身性 (Embodiment)**: 认知依赖于身体物理特征
- **嵌入性 (Embeddedness)**: 认知利用环境结构
- **延展性 (Extendedness)**: 认知延伸到外部工具
- **生成性 (Enactivism)**: 认知通过行动生成

### 3. 集体意向性深化 (Collective Intentionality Deepening 2026)

**核心框架**: 我们意图 (We-intention) 的预测加工解释

**三层架构**:
1. 共享意图层：共同目标、互补计划、响应承诺
2. 相互信念层：一阶/二阶/共同信念
3. 规范期望层：承诺、问责、集体理性

### 4. 现象学前反思理论 (Pre-reflective Phenomenology 2026)

**核心区分**:
- **前反思自我意识**: 非对象化、伴随性、直接性
- **反思自我意识**: 对象化、主题化、概念化

### 5. 主体间性预测理论 (Intersubjective Predictive Processing 2026)

**核心机制**:
- 他心预测三层架构
- 主体间预测校准
- 现象学整合（共情/直接感知/对话理解）
