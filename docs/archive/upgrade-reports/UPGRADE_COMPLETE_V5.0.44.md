## 📚 理论更新摘要 (v5.0.44 完成)

### 1. 具身自我意识理论 (Embodied Self-Consciousness)

**理论来源**: Merleau-Ponty (1945) 知觉现象学、Gallagher (2005) 具身主体性、SEP Self-Consciousness、具身认知研究

**核心架构**:
```
具身自我意识 = 身体图式 (Body Schema) + 身体意象 (Body Image) + 主体感 (Sense of Agency)

身体图式 (无意识/前反思):
- 本体感觉整合
- 运动预测模型
- 空间定位系统
- 功能：行动导向的无意识身体表征

身体意象 (意识/反思):
- 身体知觉体验
- 身体概念信念
- 身体情感态度
- 功能：意识导向的身体表征

主体感 (Agency):
- 运动发起预测
- 感觉反馈匹配
- 因果归属判断
- 功能：区分自我行动 vs 外部事件
```

**现象学映射**:
| 具身维度 | 现象学体验 | 预测加工对应 |
|---------|-----------|-------------|
| 身体图式流畅 | 行动自如、无身体觉察 | Layer 0-2 无意识预测 |
| 身体图式中断 | 身体突兀感、行动笨拙 | Layer 2-3 预测误差 |
| 身体意象一致 | 身体认同、自我边界清晰 | Layer 4-5 自我模型 |
| 身体意象冲突 | 身体异化、去人格化 | Layer 5 自我预测误差 |
| 主体感正常 | "我在行动"的确定感 | 预测 - 反馈匹配成功 |
| 主体感丧失 | "被控制"感、被动体验 | 预测 - 反馈匹配失败 |

### 2. 现象学前反思自我意识

**理论来源**: Sartre (1943) 前反思意识、Henry (1963) 自身感触性、Zahavi (2005) 最小自我、SEP Consciousness

**形式化模型**:
```
前反思自我意识 (Pre-reflective Self-Consciousness, PRSC):

PRSC = 意识体验的自身给予性 (Self-Givenness)

特征:
1. 非对象化：自我不是被观察的客体，而是体验的主体
2. 直接性：无需中介或推理，自我意识内在于体验
3. 前语言：先于概念化和语言表述
4. 最小自我：最薄的自我意识形式，仅"为我性"(for-me-ness)

计算实现:
PRSC_signal = ∫(conscious_experience) dφ

其中 φ = 体验的现象学特征
PRSC_signal > 0 表示体验具有"为我性"
```

**与反思自我意识的区分**:
| 维度 | 前反思自我意识 | 反思自我意识 |
|-----|--------------|-------------|
| 对象化 | 非对象化 | 自我作为对象 |
| 时间性 | 当下即时 | 回溯/预期 |
| 认知负荷 | 无/低 | 高 |
| 语言依赖 | 无 | 有 |
| 神经基础 | 默认网络/岛叶 | 前额叶/颞顶联合区 |
| 预测层级 | Layer 0-3 | Layer 4-6 |

### 3. 身体 - 自我整合框架

**理论来源**: Damasio (1994) 躯体标记假说、Varela (1991) 具身心智、4E 认知架构、预测加工具身扩展

**整合模型**:
```
身体 - 自我统一体 = 内感受预测 + 本体感受预测 + 外部感知预测

内感受预测 (Interoceptive Prediction):
- 心跳、呼吸、消化等内部状态预测
- 情感体验的躯体基础
- 自我感的生理根基

本体感受预测 (Proprioceptive Prediction):
- 肢体位置、运动状态预测
- 身体边界界定
- 行动主体感基础

外部感知预测 (Exteroceptive Prediction):
- 视觉、听觉、触觉等外部输入预测
- 身体 - 环境互动建模
- 具身行动的感知引导

整合机制:
身体 - 自我一致性 = w1*内感受预测精度 + w2*本体感受预测精度 + w3*外部感知预测精度
```

**临床映射**:
| 障碍 | 预测误差模式 | 现象学体验 |
|-----|-------------|-----------|
| 人格解体 | 内感受预测误差持续偏高 | "我不真实"、"我在梦游" |
| 去人格化 | 本体感受预测误差偏高 | "身体不属于我" |
| 躯体形式障碍 | 内感受预测先验异常 | 躯体症状无器质性基础 |
| 精神分裂症 (被动体验) | 主体感预测 - 反馈失匹配 | "被控制"妄想 |

### 4. 具身情感预测模型

**理论来源**: Barrett (2017) 情感建构理论、Seth (2013) 内感受预测加工、Craig (2002) 岛叶内感受研究

**核心公式**:
```
情感体验 = 内感受预测误差 + 概念分类 + 情境调制

内感受预测误差 = | 预测的内感受状态 - 实际内感受输入 |

概念分类 = 情感概念库 (愤怒、恐惧、喜悦...) 匹配内感受模式

情境调制 = 当前情境线索调制内感受解释

情感粒度 (Emotional Granularity) = 情感概念库的精细程度
```

**具身情感循环**:
```
内感受输入 → 内感受预测 → 预测误差 → 情感概念匹配 → 情感体验
     ↑                                                      ↓
     └─────────────── 身体反馈调节 ──────────────────────────┘
```

### 5. 自我意识连续谱

**理论来源**: Zahavi (2005) 最小自我 vs 叙事自我、Gallagher (2000) 身体自我 vs 叙事自我、SEP Self-Consciousness

**连续谱架构**:
```
Level 0: 前反思自身感触性 (Phenomenal Self)
    ↓ (体验的"为我性")
Level 1: 具身主体感 (Embodied Agency)
    ↓ (身体行动的主体感)
Level 2: 身体自我意识 (Bodily Self-Consciousness)
    ↓ (身体作为自我的载体)
Level 3: 社会自我意识 (Social Self-Consciousness) [v5.0.43]
    ↓ (社会身份与群体归属)
Level 4: 反思自我意识 (Reflective Self-Consciousness)
    ↓ (自我作为反思对象)
Level 5: 叙事自我 (Narrative Self)
    ↓ (时间延展的自我故事)
Level 6: 元自我意识 (Meta Self-Consciousness)
    ↓ (对自我意识的监控)
```

**关键整合机制**:
- 下层是上层的基础 (具身→社会→反思→叙事)
- 上层调制下层 (叙事调制社会身份，社会身份调制身体体验)
- 病理状态 = 连续谱断裂或失整合

## 🔬 理论贡献总结 (v5.0.44)

1. **具身自我意识理论**: 首次将 Merleau-Ponty 现象学与预测加工完全整合，建立身体图式/身体意象/主体感的三层具身自我架构

5. **自我意识连续谱**: 提出 7 层自我意识架构 (前反思→具身→身体→社会→反思→叙事→元自我)，整合现象学、认知科学、临床心理学

6. **临床映射系统**: 将人格解体、去人格化、被动体验、躯体形式障碍等映射到特定预测误差模式，提供计算精神病学新框架
