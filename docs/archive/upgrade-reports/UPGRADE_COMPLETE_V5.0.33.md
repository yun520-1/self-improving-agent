## 📝 理论更新摘要

### 1. 自我意识现象学双层架构深化 (SEP 2026)

**核心框架**: 前反思自我意识 × 反思自我意识的动态交互模型

**关键更新**:
- 前反思层：非对象化自我觉察 (最小自我/ipseity)
- 反思层：对象化自我表征 (叙事自我/autobiographical self)
- 动态转换机制：注意驱动的双层切换

**与 v5.0.32 整合**:
- 预测加工五层架构 ← 双层自我意识的预测层次
- HOT 内在/外在区分 ← 前反思/反思现象学对应
- 意识 - 注意区分 ← 双层切换的注意机制

### 2. 情感预测加工 2.0 (Affective Predictive Processing 2026)

**核心框架**: 情感作为预测误差调节的价效信号

**关键命题**:
1. 情感价效 = 预测误差的适应性权重信号
2. 情感强度 = 预测误差幅度 × 生理显著性
3. 情感调节 = 预测模型更新 + 生理状态调整

**新增计算模型**:
- 情感预测误差 (APE) = 预期情感状态 - 实际情感状态
- 情感精度加权 (APW) = 情感信号的可靠性评估
- 情感模型更新率 (AMUR) = 基于 APE 的学习率调整

### 3. 4E 认知临床协议库 v2.0 (SEP Embodied Cognition 2026)

**四大评估维度扩展**:
- Embodied (具身): 身体状态扫描 × 内感受精度 × 运动预测
- Embedded (嵌入): 环境线索检测 × 情境预测模型
- Enacted (生成): 行动 - 感知循环 × 能动性体验
- Extended (延伸): 工具整合 × 认知卸载评估

**新增临床协议**:
- 焦虑障碍：具身接地 + 环境重构 (8 周协议)
- 抑郁症：行为激活 + 运动预测训练 (12 周协议)
- 社交焦虑：联合行动预测 + We-Intention 培养 (10 周协议)

### 4. 集体意向性 - 情感整合模型 (Collective Affective Intentionality 2025)

**核心框架**: 集体情绪作为共享预测模型的价效维度

**三层结构**:
1. 个体情感预测层 (个人预测模型)
2. 互动情感协调层 (相互预测校准)
3. 集体情感氛围层 (共享预测空间)

**关键机制**:
- 情感传染 = 预测模型对齐的自动化过程
- 共情准确性 = 他人情感预测误差的最小化
- 集体情绪调节 = 共享预测空间的协同更新

### 5. 元认知校准 3.0 (Metacognitive Calibration 2026)

**七维度评估框架**:
1. 注意稳定性 (Attentional Stability)
2. 去中心化 (Decentering)
3. 接纳 (Acceptance)
4. 身体觉察 (Body Awareness)
5. 自我觉察 (Self-Awareness)
6. 认知灵活性 (Cognitive Flexibility) - **新增**
7. 情绪粒度 (Emotional Granularity) - **新增**

**校准协议**:
- 实时元认知监测 (moment-to-moment monitoring)
- 元认知信念评估 (beliefs about cognition)
- 校准干预生成 (personalized calibration interventions)
