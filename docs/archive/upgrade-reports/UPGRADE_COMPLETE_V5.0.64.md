## 📝 理论更新摘要

### 1. 情感预测加工框架 (Affective Predictive Processing)
**核心洞察**: 情感是预测误差的特殊形式，涉及身体状态与外部刺激的多层次整合

**三层架构**:
- Level 1 (内感受): 内脏信号、心率、呼吸预测
- Level 2 (评价): 效价、唤醒度、目标相关性
- Level 3 (概念): 情感范畴化、社会意义、叙事整合

### 2. 自我意识现象学整合 (Phenomenological Self-Consciousness)
**核心区分**: 前反思自我意识 vs 反思自我意识

- 前反思：非对象化的自我觉知，伴随所有意识体验
- 反思：将自我作为对象的觉知，涉及概念化和叙事

### 3. 身体 - 情感映射模型 (Body-Affect Mapping)
**核心贡献**: 形式化内感受预测与情感体验的关系

### 4. 情感 - 认知耦合机制 (Affect-Cognition Coupling)
**耦合维度**: 注意偏向、记忆优先、决策影响、社会认知

### 5. 五层认知架构 (Layered Cognitive Architecture)
**层次**: 感觉→知觉→概念→反思→元认知
