## ✨ 新增模块

### 1. 前反思自我意识增强模块 (`src/prereflective-consciousness-enhanced/`)

**功能**:
- 评估用户描述中的前反思自我意识水平
- 检测直接体验 vs 反思性描述
- 生成现象学还原引导
- 提供个性化前反思觉察练习

**核心 API**:
```javascript
// 评估前反思自我意识
const assessment = PrereflectiveConsciousnessEnhanced.assessPrereflectiveAwareness(description);

// 引导现象学还原
const guide = PrereflectiveConsciousnessEnhanced.guidePhenomenologicalReduction(description);

// 生成练习
const practices = PrereflectiveConsciousnessEnhanced.generatePrereflectivePractice(userProfile, assessment);
```

**评估维度**:
- 直接性 (Immediacy): 检测非反思性的直接体验描述
- 第一人称给定性 (First-Personal Givenness): 评估"为我性"体验
- 非对象化 (Non-Objectifying): 识别非对象化的自我关系
- 体验厚度 (Phenomenal Thickness): 分析体验的多维度丰富性

### 2. We-Intention 检测器 (`src/we-intention-detector/`)

**功能**:
- 检测用户描述中的 We-Intention (我们意向)
- 评估联合承诺强度
- 分析信任成分（认知信任 vs 规范信任）
- 生成培养集体意向的建议

**核心 API**:
```javascript
// 检测 We-Intention
const detection = WeIntentionDetector.detectWeIntention(description, context);

// 评估联合承诺
const commitment = WeIntentionDetector.assessJointCommitment(description, participants);
```

**检测维度**:
- 语言标记分析 (30% 权重)
- 承诺性质分析 (35% 权重)
- 相互依赖分析 (20% 权重)
- 规范性分析 (15% 权重)

## 📚 可操作技术

### 现象学还原练习 (Phenomenological Reduction)

**目的**: 帮助用户区分体验本身和对体验的解释

**步骤**:
1. 识别描述中的解释性/判断性语言
2. 悬置（epoché）这些判断
3. 直接描述体验的给定性
4. 对比还原前后的差异

**示例**:
- 原描述："我感到焦虑，因为我明天要演讲，这说明我不够自信"
- 还原后："我感到胸口紧绷，呼吸变浅，手心出汗，思绪快速流转"

### We-Intention 培养练习

**目的**: 增强集体意向和联合承诺

**练习**:
1. **共同目标设定**: 用"我们"而非"我"表述目标
2. **联合承诺声明**: 明确表达"我们承诺一起..."
3. **相互响应练习**: 在合作中刻意协调行动
4. **信任构建对话**: 讨论能力和承诺两个维度

### 体验厚度扩展练习

**目的**: 丰富体验的现象学维度

**6 个维度**:
1. 身体感觉
2. 情绪感受
3. 感知体验
4. 时间体验
5. 空间体验
6. 意义体验

### 前反思自我意识评估测试

```javascript
const PrereflectiveConsciousnessEnhanced = require('./src/prereflective-consciousness-enhanced');

// 测试 1: 高前反思描述
const highPrereflective = PrereflectiveConsciousnessEnhanced.assessPrereflectiveAwareness(
  '我感到胸口紧绷，呼吸变得浅而快，手心在出汗，思绪像飞转的轮子'

console.log(highPrereflective.overall.prereflectiveLevel); // 预期：'高'或'很高'

// 测试 2: 高反思描述

  '我在反思我的焦虑情绪，分析它的原因，思考这说明了什么问题'
);
console.log(highReflective.overall.reflectiveLevel); // 预期：'高'
console.log(highReflective.overall.balance); // 预期：'反思主导'
```

### We-Intention 检测测试

```javascript
const WeIntentionDetector = require('./src/we-intention-detector');

// 测试 1: 强 We-Intention
const strongWe = WeIntentionDetector.detectWeIntention(
  '我们约定好要一起完成这个项目，不能让对方失望，这是我们共同的承诺'
);
console.log(strongWe.overall.weIntentionPresent); // 预期：true
console.log(strongWe.overall.strength); // 预期：'强'或'很强'

// 测试 2: 个体承诺聚合
const individualCommitments = WeIntentionDetector.detectWeIntention(

);
console.log(individualCommitments.overall.weIntentionPresent); // 预期：false
console.log(individualCommitments.overall.type); // 预期：'个体承诺聚合'
```
