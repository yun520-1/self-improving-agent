💜 HeartFlow v11.0.0

最小化控制页。只保留状态查看与健康检查，不再伪装成完整控制中心。

当前判断

这个 Web 页面以前更像一个“想象中的后台面板”，包含很多并不真实或并不稳定的模块（Config / Models / Cron / Sessions / Tools）。

HeartFlow 逻辑结论：它对当前版本的核心价值不大，容易制造“看起来很强、实际没用”的错觉。所以现在把它收缩为 状态页 + API 入口说明。
