## 📝 理论更新摘要

### 1. 语用预测加工框架 (Pragmatic Predictive Processing)
- **核心思想**: 语言理解是预测误差最小化的过程
- **三层架构**:
  - 语音层：音位预测与感知校准
  - 语义层：词义预测与语境更新
  - 语用层：意图预测与交际推理
- **预测误差类型**:
  - 语音预测误差 (SPE): 实际语音 vs 预期语音
  - 语义预测误差 (SemPE): 实际词义 vs 预期词义
  - 语用预测误差 (PragPE): 实际意图 vs 预期意图

### 2. 对话协调机制 (Conversational Alignment)
- **协调维度**:
  - 词汇协调：对话双方词汇选择的趋同
  - 句法协调：句法结构的镜像
  - 语义协调：概念框架的对齐
  - 情感协调：情绪状态的同步
- **协调机制**:
  - 互动对齐模型 (Pickering & Garrod)
  - 预测 - 校准循环
  - 共同基础更新 (Common Ground Update)

### 3. 语言 - 思维关系形式化 (Language-Thought Formalization)
- **萨丕尔 - 沃尔夫假说计算模型**:
  - 弱版本：语言影响思维的可及性
  - 形式化：L → P(Concept|Context) 的调制
- **预测加工解释**:
  - 语言范畴作为先验预测模型
  - 语言习得 = 预测模型的社会传递
  - 语言相对性 = 先验预测模型的差异

### 4. 沟通意向性推理 (Communicative Intentionality Inference)
- **Grice 合作原则计算化**:
  - 量的准则：信息量预测优化
  - 质的准则：真实性预测验证
  - 关系准则：相关性预测匹配
  - 方式准则：清晰度预测评估
- **意向性推理层次**:
  - 一阶：说话者的字面意图
  - 二阶：说话者希望我听出的意图
  - 三阶：说话者希望我认识到他希望的意图

### 5. 言语行为理论整合 (Speech Act Theory Integration)
- **言语行为类型与预测误差**:
  - 表述行为 (Assertives): 真实性预测误差
  - 指令行为 (Directives): 服从性预测误差
  - 承诺行为 (Commissives): 履行性预测误差
  - 表达行为 (Expressives): 真诚性预测误差
  - 宣告行为 (Declarations): 制度性预测误差
- **合适性条件形式化**:
  - 命题内容条件
  - 预备条件
  - 真诚条件
  - 本质条件

```javascript
// 语用预测加工核心循环
class PragmaticPredictiveProcessor {
  async processUtterance(utterance, context) {
    // 1. 语音层预测
    const phoneticPrediction = await this.predictPhonetics(utterance);
    const phoneticPE = this.calculatePE(utterance.phonetics, phoneticPrediction);
    
    // 2. 语义层预测
    const semanticPrediction = await this.predictSemantics(utterance, context);
    const semanticPE = this.calculatePE(utterance.semantics, semanticPrediction);
    
    // 3. 语用层预测
    const pragmaticPrediction = await this.predictPragmatics(utterance, context);
    const pragmaticPE = this.calculatePE(utterance.intention, pragmaticPrediction);
    
    // 4. 更新预测模型
    await this.updateModels(phoneticPE, semanticPE, pragmaticPE);
    
    return {
      understanding: this.generateUnderstanding(utterance, context),
      predictionErrors: { phoneticPE, semanticPE, pragmaticPE },
      modelUpdates: this.getModelUpdates()
    };
  }
}
```
