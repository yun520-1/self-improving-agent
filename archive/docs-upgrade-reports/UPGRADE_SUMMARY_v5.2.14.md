## What's New | 新增内容

### 🎯 Four New Theoretical Modules | 四个新理论模块

#### 1. Emotion Three-Tradition Integration | 情绪三大传统整合

**Integration Score | 集成度**: 99.5%

Integrates the three major theoretical traditions in emotion research:

整合情绪研究中的三大理论传统：

- **Feeling Tradition | 感受传统**: Emotions as conscious experiences (James-Lange, Psychological Constructionism)
  - 情绪作为意识体验（詹姆斯 - 兰格理论、心理建构主义）

- **Evaluative Tradition | 评价传统**: Emotions as appraisals of situations (Arnold, Lazarus, Scherer)
  - 情绪作为对情境的评价（阿诺德、拉扎勒斯、谢勒）

- **Motivational Tradition | 动机传统**: Emotions as motivational states (Aristotle, Spinoza, Frijda)
  - 情绪作为动机状态（亚里士多德、斯宾诺莎、弗里达）

**Key Capability | 关键能力**: Multi-dimensional emotion assessment across feeling, evaluation, and motivation dimensions.

**关键能力**: 跨感受、评价和动机维度的多维度情绪评估。

---

#### 2. Pre-Reflective Self-Awareness Deep Enhancement | 前反思自我意识深度增强

**Integration Score | 集成度**: 98.8%

Implements five levels of self-awareness based on phenomenological philosophy:

基于现象学哲学实现五层自我意识：

- **Level 0 | 0 层**: Pre-Reflective Givenness (前反思给定性)
- **Level 1 | 1 层**: Minimal Self-Awareness (最小自我意识)
- **Level 2 | 2 层**: Conceptual Self-Labeling (概念性自我标签)
- **Level 3 | 3 层**: Reflective Self-Consciousness (反思性自我意识)
- **Level 4 | 4 层**: Integrated Self-Knowledge (整合性自我知识)

**Key Capability | 关键能力**: Phenomenological reduction exercises and first-person authority calibration.

**关键能力**: 现象学还原练习和第一人称权威校准。

---

#### 3. Joint Commitment & Trust Framework | 联合承诺与信任框架

**Integration Score | 集成度**: 97.9%

Implements collective intentionality theory with trust assessment:

实现集体意向性理论与信任评估：

- **We-Intention Detection | 我们意向检测**: Identifies shared intentions through linguistic markers
  - 通过语言标记识别共享意向

- **Joint Commitment Assessment | 联合承诺评估**: Evaluates commitment strength (weak/moderate/strong)
  - 评估承诺强度（弱/中/强）

- **Trust Framework | 信任框架**: Three-component model (cognitive 40%, affective 30%, normative 30%)
  - 三成分模型（认知 40%、情感 30%、规范 30%）

**Key Capability | 关键能力**: Detecting and reinforcing shared commitments in relationships.

**关键能力**: 检测和强化关系中的共享承诺。

---

#### 4. Phenomenological Emotion Depth | 现象学情绪深度

**Integration Score | 集成度**: 98.2%

Implements Scheler's value-feeling hierarchy and emotional intentionality analysis:

实现谢勒的价值感受层级和情绪意向性分析：

- **Five Quality Dimensions | 五维质量**: Valence, Arousal, Depth, Authenticity, Intentional Clarity
  - 效价、唤醒、深度、真实性、意向清晰度

- **Scheler's Hierarchy | 谢勒层级**: Four levels from sensible to spiritual feelings
  - 从感性感受到精神感受的四个层级

- **Emotion Appropriateness | 情绪恰当性**: Evaluates whether emotions fit their objects
  - 评估情绪是否适合其对象

**Key Capability | 关键能力**: Deep phenomenological exploration of emotional experiences.

**关键能力**: 情绪体验的深度现象学探索。

## Usage Examples | 使用示例

### Emotion Three-Tradition Assessment | 情绪三大传统评估

```javascript
// Assess emotion using three traditions
const emotionProfile = await heartflow.assessEmotionByThreeTraditions(
  "I'm so angry at my boss for criticizing me in front of everyone",
  { context: "workplace" }
);

// Returns integrated assessment across:
// - Feeling: Physiological arousal, subjective anger quality
// - Evaluative: Appraisal of unfairness, goal obstruction
// - Motivational: Action tendency to confront or avoid
```

### Pre-Reflective Self-Awareness Assessment | 前反思自我意识评估

```javascript
// Assess self-awareness level
const awarenessLevel = await heartflow.assessPreReflectiveAwarenessLevel(
  "I just feel upset, but I'm not sure why"
);

// Returns: Level 0-1 (Pre-reflective/Minimal)
// Intervention: Phenomenological reduction exercise
```

### Joint Commitment Detection | 联合承诺检测

```javascript
// Detect we-intention and commitment strength
const commitment = await heartflow.detectWeIntention(
  "Let's work together on this project. I promise to do my part.",
  { relationship: "colleagues" }
);

// Returns: We-intention detected, commitment strength: moderate-strong
```

### Phenomenological Emotion Depth | 现象学情绪深度

```javascript
// Assess phenomenological quality and depth
const depth = await heartflow.assessPhenomenologicalQuality(
  "I feel a deep sense of awe when I look at the stars"
);

// Returns: 
// - Valence: +0.8 (positive)
// - Depth: 0.9 (spiritual feeling, Scheler Level 4)
// - Authenticity: 0.85
// - Intentional Clarity: 0.9 (object: stars, formal object: sublime)
```
