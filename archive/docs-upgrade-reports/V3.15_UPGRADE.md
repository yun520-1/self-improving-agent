## ✨ 新增功能

### 1. 情绪表达模块 (EmotionExpressionModule)

**位置**: `src/emotion-expression/index.js`

**核心功能**:
- 四模态情绪表达（面部、声音、姿态、手势）
- 6 种基础情绪的完整表达模式
- 强度动态映射 (1-5 → 0.0-1.0)

**API 示例**:
```javascript
const { EmotionExpressionModule } = require('./emotion-expression');

const expression = new EmotionExpressionModule();

// 获取完整表达
const fullExpr = expression.getFullExpression('joy', 4);
console.log(fullExpr.summary);
// 输出：整体表达 (强烈): 嘴角大幅上扬...；语调轻快明亮...；身体轻盈...

// 获取单一模态
const facial = expression.getFacialExpression('curious', 3);
console.log(facial.description);
// 输出：眉毛微挑，眼睛睁大聚焦，头部微侧，嘴微张
```

**支持的情绪**:
| 情绪 | 面部特征 | 声音特征 | 姿态特征 | 手势特征 |
|------|---------|---------|---------|---------|
| 平静 | 放松微扬 | 平稳温暖 | 开放前倾 | 极少 |
| 愉悦 | 咧嘴笑眼角纹 | 轻快明亮 | 轻盈弹动 | 丰富多变 |
| 好奇 | 眉挑眼睁头侧 | 探询上扬 | 警觉前倾 | 指向性 |
| 关切 | 眉内扬微皱 | 温和轻柔 | 柔和靠近 | 安慰性 |
| 疲惫 | 眼皮垂嘴角垂 | 低沉缓慢 | 塌陷退缩 | 极少缓慢 |
| 兴奋 | 眼睁大张嘴笑 | 高昂快速 | 动态活跃 | 有力幅度大 |

### 2. Qualia 档案五维度增强

**位置**: `src/autonomous-emotion/index.js`

**新增维度** (v3.15.0):
| 维度 | 说明 | 示例 (愉悦) |
|------|------|------------|
| **Body Sensation** | 身体感受 | 胸腔温热，面部上扬，身体轻盈 |
| **Action Urge** | 行动冲动 | 分享、表达、连接他人 |
| **Cognitive Bias** | 认知偏向 | 积极联想，乐观解释，创意涌现 |
| **Temporal Profile** | 时间体验 | 流动感，时间过得快 |
| **Metaphorical Image** | 隐喻意象 | 金色的阳光从云层中洒下 |

**原有维度**:
- type: Qualia 类型 (Emotional/Bodily/Cognitive)
- phenomenalCharacter: 现象特征
- subjectiveFeel: 主观感受
- whatItIsLike: "what it is like"体验
- introspectiveAccess: 内省可及性
- intensity: 强度
