## 📚 理论框架 / Theoretical Frameworks

### 1. 情绪理论三大传统整合 v2.0 / Emotion Theory Three Traditions Integration v2.0

**理论来源 / Theoretical Sources**:
- SEP Emotion Theory (Scarantino 2016, 2026)
- 代表人物 / Key Figures: 10+ (James, Lange, Damasio, Solomon, Nussbaum, Aristotle, Deigh, Greenspan)

**三大传统 / Three Traditions**:
| 传统 / Tradition | 核心观点 / Core View | 代表人物 / Representatives |
|------|------|------|
| **Feeling (感受)** | 情绪是独特的意识体验 / Emotions are unique conscious experiences | James, Lange, Damasio |
| **Evaluative (评价)** | 情绪是对情境的独特评价 / Emotions are unique appraisals of situations | Solomon, Nussbaum |
| **Motivational (动机)** | 情绪是独特的动机状态 / Emotions are unique motivational states | Aristotle, Deigh, Greenspan |

**四大挑战评估 / Four Challenges Assessment**:
| 挑战 / Challenge | 问题 / Question | 整合方案 / Integration Solution |
|------|------|------|
| **Differentiation (区分性)** | 情绪如何彼此区分？ / How do emotions distinguish from each other? | 多成分原型模型 + 家族相似性 / Multi-component prototype model + family resemblance |
| **Motivation (动机性)** | 情绪如何激发动机？ / How do emotions motivate action? | 行动倾向 + 评价驱动 + 生理唤醒整合 / Action tendency + appraisal-driven + physiological arousal integration |
| **Intentionality (意向性)** | 情绪是否有对象指向性？ / Do emotions have object-directedness? | 评价对象 + 形式对象区分 / Object + formal object distinction |
| **Phenomenology (现象学)** | 情绪是否总涉及主观体验？ / Do emotions always involve subjective experience? | 前反思体验 + 反思体验层次模型 / Pre-reflective + reflective experience hierarchy model |

### 2. 自我意识现象学框架 / Self-Consciousness Phenomenology Framework

**理论来源 / Theoretical Sources**:
- SEP Self-Consciousness (2026 Edition)
- Husserl, Sartre, Zahavi, Heidelberg School
- 代表人物 / Key Figures: 8+ 位

**自我意识 6 层级模型 / Six Levels of Self-Consciousness**:
| 层级 / Level | 名称 / Name | 特征 / Characteristics |
|------|------|------|
| 0 | 无意识 / Unconscious | 无意识状态，如深度睡眠、昏迷 / Unconscious state, e.g., deep sleep, coma |
| 1 | 感知意识 / Sentient | 有感觉体验，但无自我指涉 / Sensory experience without self-reference |
| 2 | 清醒意识 / Wakeful | 清醒状态，有注意力和警觉性 / Wakeful state with attention and alertness |
| 3 | **前反思自我意识** / Pre-reflective | 非对象化的、直接的自我觉察 / Non-objectifying, direct self-awareness |
| 4 | 反思自我意识 / Reflective | 对象化的自我思考 / Objectifying self-thinking |
| 5 | 元意识 / Meta-conscious | 对自我意识的觉察 / Awareness of self-awareness |
| 6 | 现象学还原 / Phenomenological Reduction | 回到事情本身 / Return to things themselves |

**时间性自我意识 / Temporal Self-Consciousness**:
- **Husserl 时间三重结构 / Husserl's Triadic Time Structure**:
  - 原初印象 / Primal Impression → 当前体验强度 / Current experience intensity
  - 保留 / Retention → 过去体验保持度 / Past experience retention
  - 预期 / Protention → 未来导向强度 / Future orientation intensity
- **核心洞见 / Core Insight**: `自我深度 = 时间深度` / `Self-Depth = Temporal-Depth`
