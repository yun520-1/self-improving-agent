## 升级信息

基于 SEP 集体意向性 (Collective Intentionality) 和自我意识 (Self-Consciousness) 理论，增强 HeartFlow 的集体情绪识别、社会认同分析和关系性自我干预能力。

## 新增模块

### 1. 集体意向性模块 (`src/collective-intentionality-enhanced/`)

**理论来源**: SEP Collective Intentionality

**核心功能**:
- ✅ We-Intention 检测 (区分个体意图与集体意图)
- ✅ 集体承诺评估 (Gilbert 联合承诺理论)
- ✅ 共享目标识别 (Bratman 共享意向性)
- ✅ 集体责任归因 (Searle 集体意向性)
- ✅ 信任评估 (Schmid 信任理论)

**可代码化的逻辑**:
```javascript
// We-Intention vs I-Intention 区分
- 个体意图：I intend to X
- 集体意图：We intend to X together
- 关键特征：互赖性、共同承诺、相互响应

// 集体意向性检测维度
1. 目标共享性 (Goal Sharing)
2. 行动互赖性 (Action Interdependence)
3. 相互响应性 (Mutual Responsiveness)
4. 承诺约束性 (Commitment Binding)
5. 身份融合度 (Identity Fusion)
```

**练习技术**:
- 集体目标澄清练习 (5 步)
- 联合承诺评估工具
- 信任修复干预框架

---

### 2. 社会认同增强模块 (`src/social-identity-enhanced/`)

**理论来源**: SEP Social Identity + Tajfel & Turner 社会认同理论

**核心功能**:
- ✅ 社会认同维度评估 (6 维度)
  - 认同显著性 (Identity Salience)
  - 认同融合 (Identity Fusion)
  - 群体自尊 (Group Self-Esteem)
  - 群体效能 (Group Efficacy)
  - 关系性自我 (Relational Self)
  - 集体连续性 (Collective Continuity)
- ✅ 内外群体偏见检测
- ✅ 社会认同威胁识别
- ✅ 认同转换干预

**可代码化的逻辑**:
```javascript
// 社会认同评估框架
{
  identitySalience: 0-1,    // 该身份在当前情境的重要性
  identityFusion: 0-1,      // 个人与群体的融合程度
  groupSelfEsteem: 0-1,     // 群体带来的自尊感
  groupEfficacy: 0-1,       // 群体实现目标的能力感
  relationalSelf: 0-1,      // 关系定义自我的程度
  collectiveContinuity: 0-1 // 群体历史的连续性感
}

// 社会认同威胁类型
- 价值威胁 (Value Threat): 群体价值被质疑
- 区分威胁 (Distinctiveness Threat): 群体独特性被模糊
- 连续性威胁 (Continuity Threat): 群体历史被中断
- 真实性威胁 (Authenticity Threat): 群体成员资格被质疑
```

**练习技术**:
- 社会认同地图绘制 (4 步)
- 认同威胁应对框架
- 多重认同整合练习

---

### 3. 集体情绪模块增强 (`src/collective-emotion-enhanced/`)

**理论来源**: SEP Collective Emotion + Durkheim 集体欢腾 + Scheler 共情理论

**核心功能**:
- ✅ 集体情绪识别 (5 类型)
  - 情绪感染 (Emotional Contagion)
  - 共情情绪 (Empathic Emotion)
  - 共享情绪 (Shared Emotion)
  - 集体情绪 (Collective Emotion)
  - 群体情绪氛围 (Group Emotional Climate)
- ✅ 集体情绪强度评估
- ✅ 集体情绪效价分析
- ✅ 集体情绪干预策略

**可代码化的逻辑**:
```javascript
// 集体情绪类型区分
{
  emotionalContagion: {     // 自动模仿，无意识
    mechanism: "mimicry",
    awareness: "low",
    intervention: "awareness"
  },
  empathicEmotion: {        // 理解他人情绪
    mechanism: "perspective-taking",
    awareness: "medium",
    intervention: "regulation"
  },
  sharedEmotion: {          // 共同体验同一情绪
    mechanism: "shared-situation",
    awareness: "high",
    intervention: "co-regulation"
  },
  collectiveEmotion: {      // 群体层面的情绪
    mechanism: "collective-intentionality",
    awareness: "high",
    intervention: "collective-action"
  }
}

// 集体情绪干预策略
- 情绪命名与正常化
- 集体意义建构
- 群体凝聚增强
- 集体行动导向
```

**练习技术**:
- 集体情绪觉察练习
- 群体情绪调节技术
- 集体意义建构框架

---

### 4. 关系性自我增强 (`src/relational-self-enhanced/`)

**理论来源**: SEP Self-Consciousness + 关系性自我理论

**核心功能**:
- ✅ 关系性自我评估 (4 维度)
  - 关系依赖度 (Relational Dependence)
  - 关系自主性 (Relational Autonomy)
  - 关系边界 (Relational Boundaries)
  - 关系整合度 (Relational Integration)
- ✅ 关系身份地图
- ✅ 关系冲突分析
- ✅ 关系成长干预

**可代码化的逻辑**:
```javascript
// 关系性自我评估
{
  relationalDependence: 0-1,  // 自我定义对关系的依赖
  relationalAutonomy: 0-1,    // 关系中保持自主的能力
  relationalBoundaries: 0-1,  // 健康边界的建立
  relationalIntegration: 0-1  // 多重关系的整合
}

// 关系身份类型
- 融合型 (Fused): 边界模糊，高度依赖
- 疏离型 (Disconnected): 边界僵硬，低度连接
- 健康型 (Integrated): 边界灵活，平衡依赖与自主
```

**练习技术**:
- 关系身份地图绘制
- 边界设定练习
- 关系自主性培养

## 升级总结

v4.6.0 升级将 HeartFlow 的能力从个体层面扩展到集体/关系层面，使系统能够：

1. **理解集体意向性**: 识别 We-Intention、集体承诺、共享目标
2. **评估社会认同**: 分析身份显著性、认同融合、群体自尊
3. **识别集体情绪**: 区分情绪感染、共情、共享情绪、集体情绪
4. **支持关系性自我**: 培养关系自主性、建立健康边界、整合多重关系

这些增强使 HeartFlow 能够更好地支持用户在社交、团队、家庭等集体情境中的情绪健康和心理成长。
