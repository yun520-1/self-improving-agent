## 📝 理论更新摘要

### 1. 集体意向性理论整合
- 形式化集体意向性五层模型
- Searle/Weber/Durkheim 经典理论的计算模型整合
- 集体意向公式：CI = Σ(I_i) + α(MB) + β(EA) + γ(NC)

### 2. 群体预测加工框架
- 三层架构：个体层/互动层/集体层
- 预测同步机制：感知/预测/行动/情感同步
- 预测同步指数 (PSI): PSI = (1/N) × Σ|P_i - P_group|

### 3. 社会Φ值模型 (Social Φ)
- 计算公式：Social Φ = Σ(个体Φ_i) + α(互动整合) + β(共享表征) + γ(集体记忆)
- 测量维度：GCI/CRI/CTE
- 应用场景：组织效能/社会运动/在线社区/文化演化

### 4. 文化认知架构
- 文化作为跨代传递的预测模型集合
- 传递机制：垂直/水平/斜向传递
- 文化维度与预测风格映射

### 5. 集体意识现象学
- 现象学特征：共同在场感/经验融合/集体情感
- 集体意识四层模型
