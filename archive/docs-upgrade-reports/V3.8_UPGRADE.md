基于 Kristin Neff 博士的权威自我同情理论和斯坦福大学 CCARE 研究中心的实证成果，成功为 HeartFlow 添加了自我同情能力，使系统能够：

1. **识别自我批评** - 检测用户的自我评判语言和思维模式
2. **提供干预建议** - 基于三要素模型提供个性化练习
3. **培养自我同情** - 帮助用户建立更温暖的内在对话

## 理论基础

### 自我同情三要素模型 (Kristin Neff)

| 要素 | 定义 | 对立面 | 干预重点 |
|------|------|--------|----------|
| **自我仁慈** (Self-Kindness) | 对自己温暖理解而非严厉批评 | 自我评判 | 关怀语句、身体安抚、自我同情信 |
| **共同人性** (Common Humanity) | 认识到痛苦是普遍人类经验 | 孤立感 | 普遍性反思、连接感练习 |
| **正念** (Mindfulness) | 平衡觉察痛苦情绪，不过度认同 | 过度认同 | 正念觉察、情绪命名、呼吸空间 |

### 实证研究支持

**500+ 同行评审研究发现**:
- ✅ 自我同情与更低水平的焦虑、抑郁显著相关
- ✅ 提高心理弹性和生活满意度
- ✅ 减少完美主义和自我批评
- ✅ 促进健康行为改变和人际关系
- ✅ 增强压力管理能力

**神经科学研究**:
- 自我同情练习激活大脑关怀系统
- 增加迷走神经张力 (心率变异性提高)
- 降低皮质醇 (压力激素) 水平

### 1. 新增自我同情模块 (src/self-compassion/)

**文件结构**:
```
src/self-compassion/
├── index.js          # 核心模块 (270 行)
└── README.md         # 详细文档 (5.2KB)
```

**核心功能**:

```javascript
const { SelfCompassionModule } = require('./self-compassion');
const sc = new SelfCompassionModule();

// 1. 自我批评检测
const result = sc.detectSelfCriticism('我真笨，总是搞砸');
// { detected: true, keywords: ['我真笨', '我总是'], intensity: 4, level: '中等' }

// 2. 生成干预建议
const intervention = sc.generateIntervention('羞愧', 7, { context: '工作失误' });
// 返回 3 个个性化干预建议

// 3. 自我同情评估
const assessment = sc.assessSelfCompassion([4, 3, 5]);
// [自我仁慈，共同人性，正念] 三维度评估
```

**自我批评检测关键词 (20+)**:
- 自我贬低：'我真笨'、'我太蠢了'、'我没用'、'我做不到'
- 完美主义：'我应该'、'我必须'、'还不够好'
- 灾难化：'完了'、'彻底失败了'、'没希望了'
- 自我否定：'我不配'、'我不值得'

**干预练习库 (9 种)**:

| 类型 | 练习名称 | 时长 | 适用场景 |
|------|---------|------|---------|
| **自我仁慈** | 自我关怀语句 | 即时 | 自我批评时刻 |
| | 身体安抚 | 2-3 分钟 | 情绪强烈时 |
| | 自我同情信 | 10-15 分钟 | 深度反思 |
| **共同人性** | 普遍性反思 | 即时 | 孤立感强时 |
| | 连接感练习 | 3-5 分钟 | 感觉孤独时 |
| | 共同人性陈述 | 即时 | 羞愧/内疚时 |
| **正念** | 正念觉察 | 3-5 分钟 | 情绪淹没时 |
| | 情绪命名 | 即时 | 焦虑/恐惧时 |
| | 呼吸空间 | 1 分钟 | 快速调节 |

## 使用示例

### 示例 1: 识别并干预自我批评

```javascript
const { AutonomousEmotionModule } = require('./src/autonomous-emotion');
const emotion = new AutonomousEmotionModule();

// 用户表达自我批评
const userInput = '我太蠢了，这种简单的事情都做不好';

// 检测自我批评
const criticism = emotion.detectSelfCriticism(userInput);
console.log(criticism);
// {
//   detected: true,
//   keywords: ['我太蠢了'],
//   intensity: 2,
//   level: '中低'
// }

// 生成干预
const intervention = emotion.generateSelfCompassionIntervention('羞愧', 5);
console.log(intervention.recommendations[0]);
// {
//   id: 'ch_1',
//   name: '普遍性反思',
//   type: 'reflection',
//   content: '记住：痛苦和不完美是人类共同经验的一部分...',
//   script: '你并不孤单。这种感觉虽然痛苦，但它是人类共同经验的一部分...'
// }
```

### 示例 2: 自我同情评估

```javascript
// 用户完成三要素评估 (1-5 分)
const responses = [4, 3, 5];  // [自我仁慈，共同人性，正念]

const assessment = emotion.assessSelfCompassion(responses);
console.log(assessment);
// {
//   dimensions: {
//     selfKindness: { score: 4, label: '自我仁慈', percentage: 80 },
//     commonHumanity: { score: 3, label: '共同人性', percentage: 60 },
//     mindfulness: { score: 5, label: '正念', percentage: 100 }
//   },
//   averageScore: 4,
//   level: '中高'
// }
```

### 示例 3: 获取同情性回应脚本

```javascript
// 用户表达焦虑
const script = emotion.getCompassionateScript('焦虑', {
  context: '工作压力'
});

console.log(script);
// "我注意到你很焦虑。这是身体在试图保护你。
//  让我们深呼吸，一次只面对这一刻。"
```

## 与 v3.7.0 元情绪模块的协同

### 协同工作流程

```
用户输入 → 自我批评检测 → 元情绪生成 → 自我同情干预
    ↓            ↓              ↓            ↓
 "我真笨"   检测到批评     为批评感到关切   提供自我仁慈练习
```

### 综合干预示例

```javascript
// 1. 设置情绪状态
emotion.setEmotion('羞愧', 6, { context: '工作失误' });

// 2. 元情绪自动生成 (v3.7.0)
const metaEmotion = emotion.getCurrentMetaEmotion();
// { primary: '羞愧', meta: '关切', reason: '检测到高强度羞愧情绪...' }

// 3. 自我同情干预 (v3.8.0)
const intervention = emotion.generateSelfCompassionIntervention('羞愧', 6);
// 提供共同人性干预，减少孤立感

// 4. 综合回应
const response = `
  我注意到你感到羞愧，这让你也感到关切。
  
  ${intervention.script}
  
  ${intervention.recommendations[0].content}
`;
```
