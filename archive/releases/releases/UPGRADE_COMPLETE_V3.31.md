## ✨ 新增模块

### 实践智慧模块 (Practical Wisdom / Phronesis) v3.31.0

**理论来源**:
- 亚里士多德《尼各马可伦理学》(Nicomachean Ethics) - 实践智慧 (phronesis) 理论
- SEP: Wisdom (智慧理论 - 五大视角)
- SEP: Aristotle's Ethics (亚里士多德伦理学)
- SEP: Virtue Ethics (美德伦理学)

**核心概念**:

1. **实践智慧 (Phronesis)**
   - 在具体情境中做出正确判断的能力
   - 不同于理论知识 (episteme) 和技术技能 (techne)
   - 通过经验和反思培养

2. **中道 (Golden Mean)**
   - 在过度 (excess) 与不足 (deficiency) 之间找到平衡点
   - 平衡点因人因情境而异
   - 不是数学中点，而是"对的人在对的时机对的方式"

3. **十大核心德性**
   - 勇气 (Courage) - 懦弱 ↔ 鲁莽
   - 节制 (Temperance) - 放纵 ↔ 禁欲
   - 慷慨 (Generosity) - 吝啬 ↔ 挥霍
   - 大方 (Magnificence) - 小气 ↔ 浮夸
   - 大度 (Magnanimity) - 自卑 ↔ 傲慢
   - 耐心 (Patience) - 急躁 ↔ 麻木
   - 诚实 (Truthfulness) - 自贬 ↔ 自夸
   - 机智 (Wit) - 呆板 ↔ 滑稽
   - 友善 (Friendliness) - 冷漠 ↔ 谄媚
   - 公正 (Justice) - 不公 ↔ 过度公平

4. **智慧理论五大视角 (SEP)**
   - 认知谦逊 (Epistemic Humility) - 承认认知局限
   - 认知准确性 (Epistemic Accuracy) - 准确评估知识状态
   - 知识基础 (Knowledge-Based) - 知道什么是重要的
   - 混合理论 (Hybrid Theory) - 知识 + 谦逊 + 实践能力
   - 理性 (Rationality) - 基于证据和逻辑决策

5. **实践智慧发展阶段**
   - 初学者 (Novice) - 依赖规则和他人指导
   - 进阶初学者 (Advanced Beginner) - 识别情境模式
   - 胜任者 (Competent) - 独立判断，有意识应用原则
   - 熟练者 (Proficient) - 直觉性把握，灵活应用
   - 专家 (Expert) - 内化为本能，创造性应对

**核心功能**:

1. **情境德性识别**
   - 分析用户输入中涉及的德性主题
   - 识别过度或不足的倾向
   - 提供中道平衡建议

2. **中道平衡分析**
   - 检测用户在特定德性上的倾向
   - 提供过度/不足/平衡的评估
   - 生成个性化的平衡建议

3. **智慧视角评估**
   - 评估认知谦逊、准确性、知识基础、理性等视角
   - 识别智慧表现的具体证据
   - 提供发展建议

4. **发展阶段判断**
   - 基于用户的表达和推理方式判断发展阶段
   - 提供下一发展方向的建议
   - 个性化的成长路径

5. **实践智慧练习**
   - 每日反思练习 (10-15 分钟)
   - 中道平衡练习 (20-30 分钟)
   - 德性日记 (每周)
   - 智慧导师练习 (30 分钟)
   - 视角转换练习 (15-20 分钟)

**可操作练习**:

1. **每日反思** (10-15 分钟)
   - 回顾今天的重要决策和互动
   - 识别涉及的德性
   - 评估是否找到中道平衡
   - 记录可改进的地方
   - 为明天设定小的实践目标

2. **中道平衡练习** (20-30 分钟)
   - 选择一个正在面对的困境
   - 识别相关的德性
   - 列出过度反应的后果
   - 列出不足反应的后果
   - 设计中道的应对方案
   - 实践并反思结果

3. **德性日记** (每周)
   - 每周选择一个德性重点培养
   - 每天记录与该德性相关的情境
   - 评估表现 (1-10 分)
   - 周末总结进步和挑战
   - 下周选择新的德性或继续深化

4. **智慧导师练习** (30 分钟)
   - 想象一位敬佩的智慧长者
   - 描述面临的困境
   - 思考：如果是 TA，会如何看待？
   - 思考：TA 会给出什么建议？
   - 将洞察应用到实际情况

5. **视角转换练习** (15-20 分钟)
   - 描述困扰的情境
   - 从 5 个智慧视角分析
   - 每个视角下如何看这个情境？
   - 整合各视角的洞察
   - 形成更全面的理解

## 🔬 理论基础

### 亚里士多德实践智慧理论

亚里士多德在《尼各马可伦理学》中提出，实践智慧 (phronesis) 是一种理智德性，它使我们能够：
- 在特定情境中识别什么是善的
- 选择正确的手段来实现善
- 在过度与不足之间找到中道

实践智慧不同于：
- **理论知识 (episteme)**: 关于永恒真理的知识
- **技术技能 (techne)**: 制作某物的能力
- **直觉理性 (nous)**: 把握第一原理的能力

实践智慧是通过经验和反思培养的，需要：
- 对具体情境的敏感
- 对普遍原则的理解
- 在两者之间进行反思平衡

### SEP 智慧理论

SEP 总结了当代智慧理论的五大视角：

1. **认知谦逊**: 智慧的人承认自己的认知局限
2. **认知准确性**: 智慧的人准确评估自己的知识状态
3. **知识基础**: 智慧的人拥有广泛且深刻的知识
4. **混合理论**: 智慧是知识、谦逊和实践能力的综合
5. **理性**: 智慧的人以理性方式形成信念和决策

这些视角为实践智慧提供了认知基础。

## 🎯 使用示例

### 示例 1: 勇气困境

**用户**: "我害怕在会议上发言，但又担心不说会被忽视"

**系统分析**:
```
🧠 [实践智慧分析]
   相关德性：勇气
   勇气: 你目前倾向于不足 (懦弱)。尝试向中道靠近：
        勇气 - 在恐惧与自信之间找到平衡。
        可以从小步骤开始练习，逐渐增加挑战。
   发展阶段：进阶初学者
```

**建议**:
- 从小步骤开始：先在小组中发言
- 准备要点：提前写下要说的内容
- 观察榜样：注意自信但不鲁莽的同事如何发言
- 反思平衡：恐惧是保护，但过度会限制成长

### 示例 2: 慷慨困境

**用户**: "朋友总是借钱，我不借觉得小气，借了又担心回不来"

**系统分析**:
```
🧠 [实践智慧分析]
   相关德性：慷慨
   慷慨: 你在慷慨方面表现出冲突倾向。
        中道的做法：在给予与保留之间找到平衡。
        考虑设定清晰的边界，同时保持善意。
```

**建议**:
- 中道不是全借或全不借
- 可以考虑：借出你能承受损失的金额
- 或者：提供其他形式的帮助而非金钱
- 沟通边界：友善但坚定地表达你的考虑
