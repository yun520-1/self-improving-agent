**Source | 来源**: SEP Emotion §2 (2026) + Scarantino (2016) + Fehr & Russell (1984)

- **Feeling Tradition** (0.961 integration)
  - James-Lange constructivist phenomenology
  - Bodily feeling as perception of autonomic/motor changes
  - Distinctive phenomenological quality per emotion type
  - Intensity mapping with valence dimension (0-100 scale)

- **Evaluative Tradition** (0.958 integration)
  - Four appraisal dimensions: relevance, implications, coping potential, normative significance
  - Formal object theory: each emotion has distinctive correctness conditions
  - Rationality assessment: emotion appropriateness evaluation
  - Cross-cultural normative standards support

- **Motivational Tradition** (0.955 integration)
  - Action tendency mapping per emotion category
  - Goal relevance tracking: emotion signals goal progress/obstruction
  - Urgency modulation: priority signal for attention/action allocation
  - Behavioral disposition profiling

- **Four Theoretical Challenges Resolved | 四大理论挑战解决**:
  | Challenge | Solution | Quality |
  |-----------|----------|---------|
  | Differentiation | Three-tradition multi-dimensional profiling | 0.962 |
  | Motivation | Motivational tradition + action tendency mapping | 0.958 |
  | Intentionality | Evaluative tradition + formal object theory | 0.955 |
  | Phenomenology | Feeling tradition + constructivist account | 0.961 |

### 2. 🧠 Self-Consciousness Dual-Layer Model | 自我意识双层模型

**Integration Score | 整合评分**: 0.956 (+0.004)

**Source | 来源**: SEP Self-Consciousness §2-§4 (2026) + Shoemaker (1968) + Evans (1982) + Zahavi (2005)

- **Pre-Reflective Layer** (0.953 accuracy)
  - Heidelberg School + Zahavi immediate self-acquaintance
  - Non-objectifying self-relation
  - First-personal givenness
  - Minimal self (ipseity) detection
  - Linguistic markers for pre-reflective self-reference

- **Reflective Layer** (0.948 accuracy)
  - Shoemaker IEM (Immunity to Error through Misidentification) classification
  - Three IEM types:
    - **Bodily IEM**: Pain, proprioception (0.956 confidence)
    - **Mental IEM**: Thoughts, intentions (0.951 confidence)
    - **Agency IEM**: Actions, decisions (0.948 confidence)
  - Non-IEM types: observational and inferential self-knowledge

- **Shoemaker Integration**:
  - Self-blindness impossibility argument
  - Moore's paradox handling for belief self-ascription
  - Rationality constraint on self-knowledge

- **Evans Integration**:
  - Varieties of reference: I-thoughts and identification-free self-reference
  - Demonstrative identification: self vs. other distinction
  - Embodied self-location in objective world

  | Model | Key Insight | Integration Score |

  | **Gilbert (1989)** | Joint commitment + plural subject + demand rights | 0.958 |

- **Trust Framework** (Schmid 2013):
  - Cognitive component: Belief in partner competence/reliability (0.945)
  - Normative component: Expectation of mutual accountability (0.942)
  - Dual structure integration (0.948)

- **Collective Emotion Phenomenology** (Walther 1923 + Scheler 1954):
  - **Four Layers**:
    - L1: Awareness of other's emotion (0.952)
    - L2: Emotion about other's emotion (0.948)
    - L3: Shared emotion with mutual awareness (0.945)
    - L4: Collective emotion as group unity (0.941)
  - Scheler Miteinanderfühlen (fellow-feeling) without explicit mutual awareness
  - Linguistic markers for each layer detection

### 4. 🏃 Embodied Cognition Multi-Layer Integration | 具身认知多层整合

**Integration Score | 整合评分**: 0.954 (+0.003)

**Source | 来源**: SEP Embodied Cognition §1-§4 (2026) + Gibson (1966) + Merleau-Ponty (1945) + Varela (1991)

- **Ecological Layer** (Gibson 1966):
  - Affordance perception: direct perception without inference
  - Action possibilities from environment
  - Invariant detection in ambient optic array
  - Dynamic systems environment coupling (0.956)

- **Phenomenological Layer** (Merleau-Ponty 1945):
  - Lived body (Leib): pre-reflective bodily self-awareness
  - Motor intentionality: goal-directed movement without representation
  - Body schema: pre-conscious motor organization
  - Body image: reflective body representation (0.951)

- **Enactive Layer** (Varela 1991):
  - Sensorimotor contingencies: lawful action-perception couplings
  - Modal simulation: Barsalou perceptual symbols (multi-modal)
  - Autopoiesis: self-maintaining organizational closure (0.948)

- **Embodied Emotion Integration**:
  | Component | Ecological | Phenomenological | Enactive | Integrated Score |
  |-----------|------------|------------------|----------|------------------|
  | Bodily Feeling | Direct perception | Lived body experience | Sensorimotor simulation | 0.956 |
  | Action Tendency | Affordance perception | Motor intentionality | Enactive coupling | 0.953 |
  | Environment Coupling | Information pickup | Being-in-the-world | Structural coupling | 0.951 |
