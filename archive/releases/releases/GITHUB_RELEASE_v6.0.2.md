HeartFlow v6.0.2 is a comprehensive theoretical update that expands cross-cultural philosophical perspectives, integrates predictive processing frameworks, and refines neurophenomenological methods. This release maintains 99.9999% theoretical integration while adding 20 new academic sources and 5 new code components.

## 🔧 Technical Details | 技术细节

### System Requirements | 系统要求

- **Node.js**: >= 14.0.0
- **npm**: >= 6.0.0
- **OS**: macOS, Linux, Windows
- **Memory**: 512MB minimum, 2GB recommended

### Installation | 安装

```bash
# Clone repository | 克隆仓库
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill

# Install dependencies | 安装依赖
npm install

# Run HeartFlow | 运行 HeartFlow
npm start

# Or use CLI | 或使用 CLI
npx heartflow
```

### Quick Start | 快速开始

```javascript
const HeartFlow = require('./src/index.js');

// Initialize | 初始化
const hf = new HeartFlow({
  version: '6.0.2',
  language: 'zh-CN', // or 'en-US'
  culturalContext: 'EastAsian' // or 'Western', 'African'
});

// Analyze emotion | 分析情绪
const result = await hf.analyzeEmotion('我感到很伤心', {
  includePhenomenal: true,
  includeNarrative: true,
  includeCultural: true
});

console.log(result);
```
