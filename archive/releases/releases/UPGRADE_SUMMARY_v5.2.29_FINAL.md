### 2. Pre-reflective Tracking | 前反思追踪

**Improvement | 改进**: Three-layer architecture implementation  
**架构提升**: 三层架构实现

```
Layer Architecture | 层级架构
│
├─ Layer 0: Minimal Self / 最小自我
│  └─ First-personal givenness (50-100ms resolution) / 第一人称给定性（50-100ms 分辨率）
│
├─ Layer 1: Pre-reflective Awareness / 前反思觉察
│  └─ Non-objectifying self-awareness (200-500ms resolution) / 非对象化自我觉察（200-500ms 分辨率）
│
└─ Layer 2: Reflective Awareness / 反思觉察
   └─ Self as object of thought (1-2s resolution) / 自我作为思维对象（1-2s 分辨率）
```

### 3. Collective Intentionality | 集体意向性

**Improvement | 改进**: Trust-based we-intention detection  
**检测提升**: 信任基础我们意向检测

**Trust Model (Schmid 2013) | 信任模型（施密德 2013）**:
- **Cognitive Trust / 认知信任**: Belief that others will fulfill their part
- **Normative Trust / 规范信任**: Expectation grounded in mutual obligation

**Detection Markers | 检测标记**:
- Linguistic: "We feel...", "Our grief...", "Together we..."
- Phenomenological: Mutual empathy, shared emotional tone, collective attribution

### 4. Emotion Rationality | 情绪理性

**Improvement | 改进**: Five-dimension assessment framework  
**评估提升**: 五维度评估框架

| Dimension | Definition | Example |
|-----------|------------|---------|
| **Cognitive / 认知** | Emotion based on accurate beliefs | Fear based on real vs. imagined threat |
| **Strategic / 战略** | Emotion serves agent's goals | Anger motivates boundary-setting |
| **Appropriateness / 恰当性** | Emotion fits the situation | Grief proportionate to loss |
| **Justification / 证成性** | Emotion can be defended with reasons | Articulating why anger is warranted |
| **Consistency / 一致性** | Emotion coherent with other attitudes | Love consistent with care and respect |
