## New Theoretical Foundations | 新增理论基础

### 1. Embodied Predictive Emotion Framework | 具身预测情绪框架

**Source | 来源**: SEP Embodied Cognition + Predictive Processing + Emotion Theory Integration

**Key Insights | 核心洞见**:
- **Embodied cognition** rejects traditional computationalism: cognition is not just brain-based symbol manipulation / **具身认知**拒绝传统计算主义：认知不仅是基于大脑的符号操作
- **Predictive processing** frames perception and action as hierarchical inference / **预测加工**将感知和行动框架化为层级推理
- **Emotion episodes** are embodied predictive loops integrating interoception, appraisal, and action readiness / **情绪事件**是整合内感受、评价和行动准备的具身预测循环

**Five-Layer Embodied Prediction Model | 五层具身预测模型**:
| Layer | English | 中文 | Computational Function |
|-------|---------|------|----------------------|
| L1 | Interoceptive | 内感受层 | Body state prediction |
| L2 | Proprioceptive | 本体感受层 | Body position/orientation |
| L3 | Extroceptive | 外感受层 | Environmental affordances |
| L4 | Appraisal | 评价层 | High-level meaning |
| L5 | Social | 社会层 | Collective intentionality |

**Integration Algorithm | 整合算法**:
```javascript
function generateEmbodiedPrediction(emotionContext) {
  return {
    interoceptivePrediction: calculateBodyStatePrediction(emotionContext),
    proprioceptiveAlignment: assessBodyOrientation(emotionContext),
    environmentalAffordances: detectActionPossibilities(emotionContext),
    appraisalPrediction: generateHighLevelMeaning(emotionContext),
    socialPrediction: assessCollectiveContext(emotionContext)
  };
}
```

### 2. Temporal Self-Integration Enhancement | 时间 - 自我整合增强

**Source | 来源**: SEP Temporal Consciousness + Self-Consciousness + Husserl (1893-1917) + William James (1890)

**Key Insights | 核心洞见**:
- **Temporal consciousness** has three-fold structure: retention, primal impression, protention / **时间意识**有三重结构：滞留、原印象、前摄
- **Specious present** (William James) is the experienced "now" with temporal thickness / **显似现在**（威廉·詹姆斯）是具有时间厚度的体验"当下"
- **Self-consciousness** is temporally extended: I am the same self across time / **自我意识**是时间延伸的：我是跨时间的同一自我

**Husserlian Temporal Structure | 胡塞尔时间结构**:
```
[Retention] ← [Primal Impression] → [Protention]
   滞留              原印象              前摄
(past-holding)    (present-givenness)  (future-anticipation)
```

### 3. Emotion Rationality Module Enhancement | 情绪理性模块增强

**Source | 来源**: SEP Emotion §10 (Emotion and Rationality) + Scarantino (2016, 2026)

**Key Insights | 核心洞见**:
- Emotions can be evaluated along **five dimensions of rationality** / 情绪可以沿**五个理性维度**评估
- **Cognitive rationality**: accuracy of beliefs underlying emotion / **认知理性**：情绪 underlying 信念的准确性
- **Strategic rationality**: effectiveness in achieving goals / **战略理性**：实现目标的有效性
- **Propriety**: appropriateness to the situation / **恰当性**：对情境的适当性
- **Justifiability**: ability to provide reasons / **证成性**：提供理由的能力
- **Consistency**: coherence with other attitudes / **一致性**：与其他态度的一致性

**Rationality Assessment Framework | 理性评估框架**:
```javascript
function assessEmotionRationality(emotionEpisode) {
  return {
    cognitiveRationality: evaluateBeliefAccuracy(emotionEpisode),
    strategicRationality: evaluateGoalEffectiveness(emotionEpisode),
    propriety: evaluateSituationalAppropriateness(emotionEpisode),
    justifiability: evaluateReasonGivingCapacity(emotionEpisode),
    consistency: evaluateAttitudeCoherence(emotionEpisode)
  };
}
```

### 4. Meta-Emotion Monitoring Enhancement | 元情绪监控增强

**Source | 来源**: SEP Emotion + Metacognition + Flavell (1979) + Koriat (2007)

**Key Insights | 核心洞见**:
- **Meta-emotion** is emotion about emotion (e.g., feeling guilty about being angry) / **元情绪**是关于情绪的情绪（例如，因生气而感到内疚）
- **Metacognitive monitoring** tracks confidence and accuracy of emotion recognition / **元认知监控**追踪情绪识别的置信度和准确性
- **Calibration** aligns confidence with actual performance / **校准**使置信度与实际表现对齐

**Meta-Emotion Depth Levels | 元情绪深度层级**:
| Level | English | 中文 | Example |
|-------|---------|------|---------|
| L0 | No meta-awareness | 无元觉察 | Pure emotion episode |
| L1 | Basic awareness | 基本觉察 | "I am feeling angry" |
| L2 | Evaluative awareness | 评价性觉察 | "My anger is inappropriate" |
| L3 | Regulatory awareness | 调节性觉察 | "I should calm down" |
| L4 | Integrative awareness | 整合性觉察 | "This anger reveals my values" |
