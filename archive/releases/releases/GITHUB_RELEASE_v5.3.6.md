## 🎉 What's New | 新功能

## 🎯 Use Cases | 使用案例

### 1. Historical Emotion Analysis | 历史情绪分析

```javascript
const heartflow = require('heartflow-companion');

// Analyze emotion with Aristotelian framework
// 用亚里士多德框架分析情绪
const result = await heartflow.analyzeEmotion(
  "I feel angry about this injustice",
  { tradition: 'aristotelian', includeHistoricalContext: true }
);

console.log(result);
// {
//   emotion: 'anger',
//   tradition: 'aristotelian',
//   historicalContext: {
//     aristotelianView: 'Anger as response to slight or disrespect',
//     confidence: 0.95
//   }
// }
```

### 2. Cross-Cultural Comparison | 跨文化比较

```javascript
// Compare Western and Indian self-consciousness models
// 比较西方和印度自我意识模型
const comparison = await heartflow.compareTraditions(
  'cartesian',
  'advaita',
  'self-consciousness'
);

console.log(comparison);
// {
//   cartesian: 'Cogito ergo sum - indubitable self-existence',
//   advaita: 'Atman-Brahman identity - no separate self',
//   commonGround: 'Direct self-awareness without inference',
//   differences: ['Substantial self vs. no-self', ...]
// }
```

### 3. Collective Emotion Detection | 集体情绪检测

```javascript
// Analyze collective grief using Walther's four-layer model
// 使用瓦尔特四层模型分析集体悲伤
const collective = await heartflow.analyzeCollectiveEmotion(
  "The community mourns together",
  { model: 'walther' }
);

console.log(collective.layers);
// {
//   layer1: 'Shared experience detected',
//   layer2: 'Mutual empathy detected',
//   layer3: 'Identification detected',
//   layer4: 'Mutual awareness partial',
//   collectiveScore: 0.87
// }
```
