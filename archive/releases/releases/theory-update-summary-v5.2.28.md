### 2. Self-Consciousness: Historical-Contemporary Synthesis | 自我意识：历史 - 当代综合

**Source | 来源**: SEP Self-Consciousness (2026) + Descartes (1641) + Locke (1700) + Kant (1787) + Aquinas (13th c.) + Fichte (1794) + Husserl (1913) + Sartre (1937)

**Historical Lineage Integration | 历史谱系整合**:

```
Ancient/Medieval | 古代/中世纪
├─ Aristotle: Self-awareness through cognizing other things / 通过认知他物觉察自我
├─ Aquinas: Minimal self-awareness (mere presence) vs reflective / 最小自我意识（仅存在）与反思
└─ Avicenna Flying Man: Self-awareness without sensory input / 无感官输入的自我意识

Early Modern | 早期现代
├─ Descartes: Cogito ergo sum - indubitable self-existence / 我思故我在 - 不可怀疑的自我存在
├─ Locke: Intuitive knowledge of own existence + personhood definition / 自身存在的直觉知识 + 人格定义
├─ Hume: No impression of self - bundle theory challenge / 无自我印象 - 束理论挑战
└─ Kant: Transcendental apperception - "I think" must accompany all representations / 先验统觉 - "我思"必须伴随所有表象

Post-Kantian | 后康德
├─ Fichte: Immediate self-acquaintance (pre-reflective) / 直接自我熟识（前反思）
├─ Hegel: Recognition through other subjects / 通过他者主体的承认
└─ Husserl-Sartre: Pre-reflective self-awareness without transcendental ego / 前反思自我意识无先验自我
```

**Integration Score | 整合分数**: 0.945 (↑ from 0.935 in v5.2.27)

**Key Enhancements | 关键增强**:

1. **IEM (Immunity to Error through Misidentification) Calibration | IEM（免疫误认错误）校准**
   - Shoemaker (1968): First-person authority in emotion self-ascriptions / 情绪自我归因中的第一人称权威
   - Application: When user says "I feel sad", cannot be wrong about who feels / 应用：当用户说"我感到悲伤"，不可能在谁在感受上出错

2. **Pre-reflective vs Reflective Self-Consciousness | 前反思与反思自我意识**
   - Pre-reflective: Immediate, non-objectifying self-awareness (Sartre/Heidelberg School) / 前反思：直接、非对象化自我觉察
   - Reflective: Taking self as object of thought / 反思：将自我作为思维对象
   - Application: Emotion episodes tracked at both levels / 应用：情绪事件在两个层面追踪

3. **Minimal vs Reflective Self-Awareness (Aquinas) | 最小与反思自我觉察（阿奎那）**
   - Minimal: "Mere presence of mind suffices" / 最小："仅心灵存在即足够"
   - Reflective: Awareness of essence through cognizing other things / 反思：通过认知他物觉察本质
   - Application: Basic emotion feeling requires minimal; complex appraisal requires reflective / 应用：基本情绪感受需要最小；复杂评价需要反思

### 3. Collective Intentionality: Irreducibility + Individual Ownership | 集体意向性：不可还原性 + 个体所有权

**Source | 来源**: SEP Collective Intentionality (2026) + Searle (1990) + Schmid (2013, 2026) + Walther (1923) + Scheler (1954 [1912]) + Bratman (1999) + Gilbert (1990)

**Core Philosophical Claims | 核心哲学主张**:

| Claim | Definition | HeartFlow Application |
|-------|------------|----------------------|
| **Irreducibility / 不可还原性** | Collective intentionality ≠ sum of individual intentions (even with common knowledge) / 集体意向性 ≠ 个体意向之和（即使有共同知识） | We-grief ≠ my-grief + your-grief + mutual knowledge / 我们悲伤 ≠ 我的悲伤 + 你的悲伤 + 相互知识 |
| **Individual Ownership / 个体所有权** | Each participant has own mind; no collective super-brain / 每个参与者有自己的心灵；无集体超级大脑 | Naturalist version: individual brains not integrated / 自然主义版本：个体大脑不整合 |
| **Collective Attribution / 集体归属** | Intentions attributed to individuals collectively, not distributively / 意向被集体地而非分配地归属给个体 | "We intend" = you and I together intend / "我们意向" = 你和我一起意向 |

**Integration Score | 整合分数**: 0.93 (↑ from 0.92 in v5.2.27)

**Joint Commitment Structure | 联合承诺结构**:

```
Joint Commitment | 联合承诺
├─ Mutual Obligation / 相互义务
│  └─ Each participant obligated to emotional attunement / 每个参与者有义务进行情绪协调
├─ Collective Attribution / 集体归属
│  └─ Obligation belongs to participants collectively / 义务属于参与者集体
└─ Trust-Based Bonding / 信任基础联结
   └─ Cognitive + Normative components (Schmid 2013) / 认知 + 规范成分（施密德 2013）
```

**Phenomenological Controversy Resolution | 现象学争议解决**:

| Position | Claim | HeartFlow Synthesis |
|----------|-------|---------------------|
| **Walther (1923)** | Shared experience requires: (i) both experience x, (ii) mutual empathy, (iii) mutual identification, (iv) mutual awareness of identification / 共享体验需要：(i) 双方体验 x，(ii) 相互共情，(iii) 相互认同，(iv) 相互觉察认同 | ✅ Integrated: Four-layer detection with infinite regress prevention / 已整合：四层检测防止无限递归 |
| **Scheler (1954)** | Shared attitude is numerically identical state across many minds (not combination) / 共享态度是跨多个心灵的数值同一状态（非组合） | ✅ Integrated: Irreducibility claim with individual ownership constraint / 已整合：不可还原性主张与个体所有权约束 |
| **Schmid (2013)** | Trust combines cognitive + normative components / 信任结合认知 + 规范成分 | ✅ Enhanced: Trust-based we-intention detection / 增强：信任基础我们意向检测 |
