## 🎉 What's New | 新功能

### English

HeartFlow Companion v5.2.41 expands the Collective Emotion Phenomenology framework with comprehensive integration of Scheler's emotional contagion models, Walther's four-layer shared experience assessment, and Schmid's trust-based collective intentionality. This release enables advanced detection of We-Intentions, expanded collective emotion categories, and deeper integration of phenomenological shared experience with predictive processing frameworks.

**Key Highlights:**

1. **Schelerian Emotional Contagion Model** - Integrated Scheler's (1954 [1912]) theory of direct emotional sharing, affective contagion, and normatively-grounded collective emotions with five new collective emotion categories.

2. **Walther's Four-Layer Shared Experience** - Implemented Gerda Walther's (1923) four-layer model: Parallel Experience → Mutual Empathy → Mutual Identification → Recursive Mutual Awareness, with relationship-specific assessment metrics.

3. **Schmid's Trust-Based Collective Intentionality** - Added Schmid's (2013) trust model as the foundation of We-Intention, with dual cognitive (reliance) and normative (expectation) components.

4. **Enhanced We-Intention Detection** - Improved linguistic marker analysis (plural pronouns, joint action verbs, commitment language) achieving 94% detection accuracy with combined markers.

5. **Collective Emotion Categories** - Expanded to five new types: Collective Grief, National Enthusiasm, Shared Joy, Collective Anger, and Group Pride, each with prototype scoring and Schelerian type classification.

### 中文

心流伴侣 v5.2.41 扩展了集体情绪现象学框架，全面整合了舍勒的情绪感染模型、瓦尔特的四层共享体验评估、以及施密德基于信任的集体意向性。此版本支持先进的我们意向检测、扩展的集体情绪类别、以及现象学共享体验与预测加工框架的深度整合。

**核心亮点：**

1. **舍勒式情绪感染模型** - 整合舍勒 (1954 [1912]) 的直接情绪共享理论、情感感染、以及规范基础的集体情绪，新增五种集体情绪类别。

2. **瓦尔特四层共享体验** - 实现格尔达·瓦尔特 (1923) 的四层模型：平行体验 → 相互共情 → 相互认同 → 递归相互意识，具有关系特异性评估指标。

3. **施密德信任基础集体意向性** - 添加施密德 (2013) 信任模型作为我们意向的基础，具有双重认知 (依赖) 和规范 (期望) 成分。

4. **增强我们意向检测** - 改进语言标记分析 (复数代词、联合行动动词、承诺语言)，组合标记检测准确率达 94%。

5. **集体情绪类别** - 扩展至五种新类型：集体悲伤、民族热情、共享喜悦、集体愤怒、群体自豪，每种均有原型评分和舍勒类型分类。
