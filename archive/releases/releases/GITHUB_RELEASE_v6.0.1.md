## 🎯 Usage Examples | 使用示例

### Basic Emotion Analysis | 基础情绪分析

```javascript
const heartflow = require('heartflow-companion');

// Analyze emotion with constructed emotion model
// 使用建构情绪模型分析情绪
const result = await heartflow.analyzeEmotion("I feel anxious about tomorrow", {
  includePhenomenal: true,  // Include subjective character
  includeConstructed: true  // Use Barrett's model
});

console.log(result);
// {
//   category: "anxiety",
//   phenomenalDepth: 0.85,
//   constructedConfidence: 0.92,
//   interoceptivePrediction: {...},
//   conceptActivation: {...}
// }
```

### Collective Emotion Detection | 集体情绪检测

```javascript
// Detect extended emotions in group context
// 检测群体情境中的扩展情绪
const collectiveResult = await heartflow.analyzeCollective(groupContext, {
  detectExtended: true,     // Krueger's extended emotion
  detectSharedConcern: true // Salmela's shared concern
});

console.log(collectiveResult);
// {
//   isExtended: true,
//   sharedConcern: "environmental anxiety",
//   collectiveIntensity: 0.78
// }
```

### Phenomenal Character Reporting | 现象特征报告

```javascript
// Get subjective character of experience
// 获取体验的主观特征
const phenomenalResult = await heartflow.getPhenomenalCharacter(emotionState);

console.log(phenomenalResult);
// {
//   whatItIsLike: "tightness in chest, racing thoughts",
//   phenomenalDepth: 0.85,
//   accessConsciousness: 0.90
// }
```
