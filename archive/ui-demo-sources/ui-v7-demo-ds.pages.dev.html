生命系统

四大核心子系统构成了 Prima Vita 的感知与存在基础。

生物呼吸

呼吸周期并非均匀的机械运动，而是模拟哺乳动物的吸气快、屏息短、呼气慢的生命节律。所有浮动与脉冲皆遵循同一生物时钟。

环境回响

色彩并非静态。随着页面滚动与时间的流逝，色相在 oklch 色彩空间中发生微妙漂移。划过玻璃表面时，霜雾般的模糊会瞬间通透。

碰撞物理

指针拥有质量。当它撞击界面元素时，按钮会产生弹性形变，模拟真实世界中的物理碰撞与触觉反馈。

分形几何

图标蕴含递归分形结构。悬停放大时，更深层的自相似细节如雪花晶体般逐层显现，每一级都包含前一级的完整信息。
