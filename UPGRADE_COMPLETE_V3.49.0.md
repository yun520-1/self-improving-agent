### 2. 情绪 - 行动一致性评估

评估三个维度：
- ✅ **自动性**: 是否为自动化情绪反应
- ✅ **价值一致性**: 是否符合个人价值观
- ✅ **适应性**: 是否有利于长期福祉

四种对齐状态：
- **最优**: 既适应又符合价值
- **适应但不一致**: 适应但违背价值
- **一致但不适应**: 符合价值但方式不健康
- **不一致**: 既不适应也不符合价值

### 3. 实践推理中的情绪作用

| 作用类型 | 说明 | 示例 |
|---------|------|------|
| **提供理由** | 情绪本身作为行动理由 | "因为害怕所以逃避" |
| **权重影响** | 影响不同理由的权重 | 恐惧增强安全理由 |
| **情境框定** | 框定对情境的理解 | 愤怒框定为冒犯 |
| **行动承诺** | 承诺于特定行动路径 | 愤怒承诺于对抗 |

### 4. 理性审查工具

**STOP 技术**:
- **S**top - 暂停
- **T**ake breath - 呼吸
- **O**bserve - 观察
- **P**roceed - 选择行动

**智慧审查五问**:
1. 我的情绪是否基于准确评估？
2. 有其他更智慧的回应方式吗？
3. 这个行动符合我的长期价值观吗？
4. 如果情绪强度减半，我会不同选择吗？
5. 这个行动会加强还是削弱我重视的关系？

## 🔧 使用示例

```javascript
const { EmotionActionModule } = require('./src/emotion-action');
const emotionAction = new EmotionActionModule();

// 分析行动倾向
const analysis = emotionAction.analyzeActionTendency('anger', 4);
// → { primaryTendency: 'attack', actionUrge: '移除冒犯源', ... }

// 评估一致性
const alignment = emotionAction.assessEmotionActionAlignment(
  'anger', 
  '大声争吵',  // 实际行为
  '冷静沟通'   // 价值行为
);
// → { alignment: 'misaligned', suggestions: [...] }

// 实践推理分析
const reasoning = emotionAction.analyzePracticalReasoning(
  'anger',
  '同事抢了我的功劳',
  '当众对峙'
);
// → { emotionReason, weightingEffect, framingEffect, commitment, rationalReview }
```
