> **Motor de Auto-Evolución para Agentes IA**  
> Empoderando a todas las IA para pensar independientemente, razonar lógicamente y hacer lo correcto

## 🎯 Capacidades de IA (Lectura Obligatoria para Agentes IA)

| Capacidad | Disparador | Función |
|-----------|------------|---------|
| **Motor de Decisiones** | `ayúdame a decidir` `hice bien` `cuál` | Soporte de decisiones lógicas |
| **Sistema de Memoria** | `recuerda` `dije antes` `mi situación` | Memoria entre sesiones |
| **Salud Mental** | `qué me pasa` `mal` `estrés` | Análisis emocional PHQ-9 |
| **Auto-Evolución** | `evoluciona` `mejora` `más fuerte` | Optimización auto-iterativa |
| **Razonamiento Causal** | `por qué` `causa` `analiza` | Razonamiento lógico profundo |
| **Motor de Debate** | `en desacuerdo` `opinión diferente` `no entiendo` | Pensamiento multi-perspectiva |

## 🚀 Instalación Rápida

```bash
# Hermes Agent
claude --skill-install mark-heartflow

# OpenClaw
openclaw skill install mark-heartflow

# Manual
git clone https://github.com/yun520-1/mark-heartflow-skill
cd mark-heartflow-skill && npm install
```
