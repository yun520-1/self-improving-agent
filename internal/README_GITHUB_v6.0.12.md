## 🚀 Installation | 安装

### Prerequisites | 前置要求

- Node.js >= 14.0.0
- npm or yarn

### Install | 安装

```bash
# Clone repository | 克隆仓库
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill

# Install dependencies | 安装依赖
npm install

# Run the companion | 运行伴侣
npm start

# Run API server | 运行 API 服务器
npm run api

# Run tests | 运行测试
npm test

# Run demo | 运行演示

```

## 💻 Usage | 使用

### Basic Usage | 基本使用

```javascript
const HeartFlow = require('heartflow-companion');

const companion = new HeartFlow({
  identity: '小虫子 · 严谨专业版',
  language: 'zh-CN',
  tbgEnforcement: true,
  personalityTracking: true,
  aiValuesEnforcement: true
});

// Analyze emotion | 分析情绪
const result = await companion.analyzeEmotion(text, options);

// Get personality state | 获取人格状态
const personality = await companion.getPersonalityState();

// Get TBG scores | 获取真善美分数
const tbg = await companion.getTBGScores();
```

### API Endpoints | API 端点

```
POST /api/analyze          # Analyze emotion | 分析情绪
GET  /api/personality      # Get personality state | 获取人格状态
GET  /api/tbg             # Get TBG scores | 获取真善美分数
GET  /api/state           # Get system state | 获取系统状态
```
