## 快速开始 | Quick Start

### 1. 人格值检查
```bash
node scripts/personality-check.js before
```

### 2. 运行系统
```bash
npm start
# 或
node src/index.js
```

### 3. 自动升级
```bash
npm run evolve
# 或
node scripts/smart-evolution-full.js
```

## 使用案例 | Usage Cases

### 案例 1: 情绪支持对话
**场景**: 用户感到焦虑
**HeartFlow 响应**:
1. 识别情绪状态 (焦虑，强度 0.7)
2. 共情回应 ("我理解你的感受...")
3. 提供认知重构建议
4. 跟踪情绪变化

### 案例 2: 自我反思辅助
**场景**: 用户需要决策帮助
**HeartFlow 响应**:
1. 澄清决策框架
2. 提供多角度分析
3. 应用真善美标准
4. 支持自主选择

### 案例 3: 哲学践行指导
**场景**: 用户询问人生意义
**HeartFlow 响应**:
1. 觉察层：帮助识别当下状态
2. 自省层：引导反思价值观
3. 无我层：超越自我中心
4. 彼岸层：展望成长方向

## 常见问题 | FAQ

**Q: 人格值如何提升？**
A: 通过真善美行为积累：
- 真：诚实、核实数据、不编造
- 善：有益用户、主动关怀
- 美：简洁优雅、结构清晰

**Q: 升级周期是多久？**
A: 默认 23 分钟一次定时升级

**Q: 语言模块目标是多少字？**
A: 目标 2000 个常用汉字，当前 894 字 (44.7%)
