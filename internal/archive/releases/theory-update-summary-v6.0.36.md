- **HeartFlow 应用**: 系统对时间敏感决策使用满意化

- **Representativeness heuristic**: Judge by similarity to prototype
  - **代表性启发式**: 通过与原型相似性判断

- **Reference dependence**: Evaluation relative to reference point
  - **参考依赖**: 相对于参考点评估
- **Probability weighting**: Overweight small probabilities, underweight large
  - **概率加权**: 高估小概率，低估大概率
- **HeartFlow Application**: System accounts for user loss aversion in recommendations
- **HeartFlow 应用**: 系统在建议中考虑用户的损失厌恶
