## New Theories Integrated | 新增整合理论

### 1. Temporal Consciousness (时间意识)

**Source | 来源**: Stanford Encyclopedia of Philosophy (SEP)  
**Entry | 条目**: "Temporal Consciousness"  
**URL**: https://plato.stanford.edu/entries/consciousness-temporal/

**Key Concepts | 核心概念**:

| Concept | Description |
|---------|-------------|
| Cinematic Model | 意识由静态"快照"组成，类似电影帧序列 |
| Retentional Model | 意识本身是瞬间的，但包含对最近的过去的表征 (retentions) |
| Extensional Model | 意识 episode 本身在时间上延伸，能够容纳变化和持续 |
| Specious Present | James 提出的"似是而非的现在"——体验上具有短暂持续时间的现在 |
| Paradox of Temporal Awareness | 意识必须延伸到时间才能体验变化，但又似乎只能 awareness 当下 |

**Integration Points | 整合点**:
- 增强 HeartFlow 的时间体验模块
- 为记忆系统提供现象学基础
- 支持情绪的时间维度 (预期、后悔、希望)
- 改进自主决策的时间一致性检查

**Source | 来源**: Stanford Encyclopedia of Philosophy (SEP)  
**Entry | 条目**: "Decision Theory"  
**URL**: https://plato.stanford.edu/entries/decision-theory/

**Key Concepts | 核心概念**:

| Concept | Description |
|---------|-------------|
| Preference Ordering | 理性偏好的完备性和传递性公理 |
| Utility Measures | 偏好的效用测度表示 |
| Expected Utility | 期望效用最大化原则 |
| Money Pump Argument | 对违反传递性偏好的工具理性论证 |
| Revealed Preference | 偏好作为选择行为的表征 vs 作为解释选择的心理态度 |

**Integration Points | 整合点**:
- 改进 HeartFlow 的自主决策引擎

### 3. Virtue Ethics (德性伦理学)

**Source | 来源**: Stanford Encyclopedia of Philosophy (SEP)  
**Entry | 条目**: "Virtue Ethics"  
**URL**: https://plato.stanford.edu/entries/ethics-virtue/

**Key Concepts | 核心概念**:

| Concept | Description |
|---------|-------------|
| Arête (德性/卓越) | 优秀的性格特质，根深蒂固的倾向 |
| Phronesis (实践智慧) | 道德或实践智慧，辨别正确行动的能力 |
| Eudaimonia (幸福/繁荣) | 人类繁荣或幸福，德性伦理的终极目标 |
| Full vs Continence | 完全德性 (无内心冲突) vs 节制 (需要控制欲望) |
| Multi-track Disposition | 德性是多轨道的倾向，影响注意、期望、价值、感受、选择、行动和反应 |

**Integration Points | 整合点**:
- 深化人格值系统的伦理基础
- 为真善美标准提供哲学支撑
- 支持道德品格的渐进发展模型
- 增强自我改进的规范性框架
