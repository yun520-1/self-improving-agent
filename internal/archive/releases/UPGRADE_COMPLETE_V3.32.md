## ✨ 新增模块

### 关系性自我模块 (Relational Self) v3.32.0

**理论来源**:
- SEP: Self-Consciousness (Phenomenological Approaches)
- SEP: Collective Intentionality
- SEP: Phenomenology
- 现象学传统：Husserl, Heidegger, Merleau-Ponty, Sartre, Zahavi
- 共情理论：Scheler, Walther, Stein
- 社会认同理论：Tajfel & Turner

**核心概念**:

1. **前反思自我意识 (Prereflective Self-Consciousness)**
   - 体验的"为我性"(for-me-ness) - 体验以第一人称方式被给予
   - 非对象化的自我觉知 - 不需要将自我作为对象来觉知
   - 内隐性 - 体验发生时自动伴随的自我觉知
   - 基础性 - 所有反思性自我意识的前提
   - 关键洞见：除非一个心理过程是前反思地自我意识的，否则它就不会有现象感受质 (what-it-is-like)

2. **关系性自我 (Relational Self)**
   - 个体能动性 - 作为独立行动者的自我感
   - 关系认同 - 在关系中定义的自我
   - 集体归属 - 作为群体成员的自我感
   - 共情联结 - 与他人情感共鸣的能力
   - 核心观点：自我不是孤立的实体，而是在与他人的关系中动态构成的

3. **共情层次 (Empathy Levels)**
   - 情绪传染 (Level 1) - 自动捕捉并同步他人的情绪状态
   - 情感共鸣 (Level 2) - 感受到他人的情感，但保持自我 - 他人区分
   - 认知共情 (Level 3) - 理解他人的观点和感受的原因
   - 共情关怀 (Level 4) - 对他人的福祉产生关心并采取行动
   - 认同融合 (Level 5) - 自我与他人/群体的边界暂时模糊

4. **集体意向性 (Collective Intentionality)**
   - 我们意图 (We-intention) - 以"我们"为主语的意向状态，不可还原为个体意图的总和
   - 共享体验 - 共同经历同一事件或情感
   - 联合行动 - 协调行动以实现共同目标
   - 集体情绪 - 群体中传播和放大的情绪
   - 核心洞见：集体意向性不是个体意向性 + 共同知识，而是不可还原的集体现象

5. **自我 - 他人关系的健康模式**
   - 平衡的边界 - 既能保持自我完整性，又能与他人深度联结
   - 灵活的融合 - 根据情境需要调整自我 - 他人边界
   - 相互承认 - 既承认他人的主体性，也被他人承认

6. **关系性自我的发展阶段**
   - 自我中心 (Egocentric) - 难以区分自己与他人的视角
   - 社会中心 (Sociocentric) - 自我定义主要来自群体归属
   - 个体化 (Individuated) - 发展出独立的自我认同
   - 整合 (Integrative) - 整合个体自主与关系联结
   - 超越 (Transcendent) - 自我感扩展到更广泛的共同体

**核心功能**:

1. **关系情境分析**
   - 检测用户输入中的关系主题 (集体意向性、共情、自我边界、关系联结、自我认同)
   - 评估共情水平 (1-5 级)
   - 识别关系模式 (过度分离/过度融合/平衡整合)
   - 判断发展阶段

2. **共情水平评估**
   - 基于语言模式自动评估用户的共情深度
   - 深度共情指标：「我能感受到」「我理解你的」「我体会到」
   - 认知共情指标：「我明白为什么」「我能理解」「从你的角度看」
   - 情绪传染指标：「我也感到」「让我也」「一起」

3. **关系模式识别**
   - 过度分离模式：「不需要」「我自己」「别管我」
   - 过度融合模式：「都是我的错」「我必须」「都是为了」
   - 平衡整合模式：「我们可以」「一起...但也」「理解...同时」

4. **个性化建议生成**
   - 基于共情水平的建议
   - 基于关系模式的建议
   - 基于发展阶段的建议

5. **实践练习库**
   - 健康边界练习 (15-20 分钟)
   - 共情倾听练习 (10-15 分钟)
   - 视角转换练习 (20-30 分钟)
   - 我们意图练习 (15-20 分钟)
   - 关系反思日记 (10-15 分钟)
   - 健康认同融合练习 (20-30 分钟)

## 🔬 理论基础

### SEP 现象学自我意识理论

现象学传统 (Husserl, Heidegger, Merleau-Ponty, Sartre, Zahavi) 提出，意识总是伴随着一种前反思的自我意识。这种自我意识：
- 不是反思性的第二阶认知
- 不是将自我作为对象的观察
- 而是体验本身的内在结构
- 是"为我性"(for-me-ness)的体现

关键论证：
- **无限回归论证**: 如果意识需要更高阶的意识才能成为意识，会导致无限回归
- **解释空虚论证**: 两个无意识的心理状态的关系如何产生有意识的体验？
- **现象学描述**: 我们的体验总是以第一人称方式被给予

### SEP 集体意向性理论

集体意向性理论研究"我们意图"(we-intention)的本质。核心问题：
- 集体意向性是否可还原为个体意向性 + 共同知识？
- 如果是不可还原的，它如何与个体意向自主性兼容？

主要理论立场：
- **Searle**: 集体意向性是原初的，不可还原的生物学能力
- **Bratman**: 集体意向性可还原为相互响应的个体意向
- **Gilbert**: 集体意向性涉及联合承诺 (joint commitment)
- **Tuomela**: 我们 - 模式 (we-mode) vs 我 - 模式 (I-mode)

现象学贡献 (Scheler, Walther):
- **Scheler**: 共享体验不是个体体验的组合，而是真正的"一"
- **Walther**: 共享体验需要共情、认同和相互意识

### 共情理论 (Scheler, Walther, Stein)

现象学共情理论区分了不同层次的共情：
1. **情绪传染**: 前反思的情绪同步
2. **情感共鸣**: 感受他人的感受，但保持区分
3. **认知共情**: 理解他人的观点
4. **共情关怀**: 关心并采取行动
5. **认同融合**: 暂时的自我 - 他人边界模糊

关键洞见：共情不是模拟或推理，而是直接感知他人的体验。

## 🎯 使用示例

### 示例 1: 孤独感

**用户**: "我感到很孤独，即使身边有人也觉得没人理解我"

**系统分析**:
```
💞 [关系性自我分析]
   关系主题：关系联结、自我边界
   共情水平：情感共鸣 (Level 2)
   关系模式：过度分离
   说明：可能过度强调自我边界，回避亲密联结
   发展阶段：个体化
   建议:
   • 关系联结：尝试适度展示脆弱，分享你的真实感受
   • 整合发展：探索如何在保持独立的同时深化关系联结
```

**建议**:
- 练习：选择一个信任的人，分享一件你平时不会说的事
- 反思：孤独感背后的需要是什么？(被理解？被接纳？归属感？)
- 行动：参加一个你有共同兴趣的群体活动

### 示例 2: 关系边界困扰

**用户**: "我朋友总是向我倾诉负面情绪，我听得很累但不知道该怎么拒绝"

**系统分析**:
```
💞 [关系性自我分析]
   关系主题：自我边界、共情
   共情水平：共情关怀 (Level 4)
   关系模式：过度融合
   说明：可能过度承担他人责任，自我边界模糊
   发展阶段：整合
   建议:
   • 自我边界：练习区分：这是谁的感受？谁的需要？谁的责任？
```

**建议**:
- 边界练习：温和而坚定地表达你的限制
- 示例回应："我很关心你，但现在我没有足够的能量来倾听。我们可以换个时间吗？"
- 反思：帮助他人的同时，如何照顾自己的需要？

### 示例 3: 集体归属

**用户**: "我们团队一起完成了这个项目，感觉特别有成就感"

**系统分析**:
```
💞 [关系性自我分析]
   关系主题：集体意向性
   共情水平：认同融合 (Level 5)
   关系模式：平衡整合
   说明：表现出健康的自我 - 他人平衡
   发展阶段：整合
```

**建议**:
- 庆祝：和团队一起庆祝这个共同成就
- 反思：在这次合作中，你贡献了什么？从他人那里学到了什么？
- 延续：思考如何将这种集体意向性应用到未来的项目中
