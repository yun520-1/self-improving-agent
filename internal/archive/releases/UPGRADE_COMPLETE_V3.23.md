## 🧠 核心理论框架

### 1. 社会认同 (Social Identity) [Tajfel & Turner 1979]

**核心观点**: 社会认同是个体自我概念中源自群体成员身份的部分

| 要素 | 说明 |
|------|------|
| **社会分类** (Social Categorization) | 将自我和他人归类为群体成员 |
| **社会认同** (Social Identification) | 采纳群体身份作为自我概念的一部分 |
| **社会比较** (Social Comparison) | 通过群体间比较获得积极区分 |

**自我分类理论** (Turner et al. 1987):
- **元对比原则**: 组内差异 < 组间差异
- **规范适配度**: 个体特征与群体原型的匹配
- **比较适配度**: 群体分类在情境中的适用性

**认同测量指标** (Leach et al. 2008):
- 中心性 (Centrality): 认同在自我概念中的重要程度
- 满意度 (Satisfaction): 对群体成员身份的满意程度
- 团结度 (Solidarity): 与群体其他成员的联结感
- 组内联结 (In-group Ties): 感知与群体成员的相似性

### 2. 认同融合 (Identity Fusion) [Swann et al. 2009, 2012]

**核心观点**: 个人认同与集体认同的深度整合状态

**关键特征**:
- **自我 - 群体重叠**: 个人自我与集体自我的边界渗透
- **代理性感**: 个人能动性代表群体
- **不可逆性**: 融合后难以逆转
- **生理唤醒**: 为群体行动时的生理反应
- **群体利益优先**: 群体利益高于个人利益

**融合类型**:
- 完全融合 (fusionLevel ≥ 0.8)
- 强融合 (fusionLevel ≥ 0.6)
- 中等融合 (fusionLevel ≥ 0.4)
- 弱融合 (fusionLevel ≥ 0.2)
- 无融合 (fusionLevel < 0.2)

**行为预测**:
- 亲群体行为倾向
- 自我牺牲意愿 (fusionLevel > 0.8)
- 认同防御倾向 (fusionLevel > 0.6)

### 3. 关系性自我 (Relational Self) [Zahavi 2015]

**核心观点**: 自我概念中包含与他人的关系，形成"我们感"

**现象学自我理论特征**:
- **最小自我** (Minimal Self): 前反思自我意识
- **叙事自我** (Narrative Self): 扩展的自我概念
- **社会自我** (Social Self): 关系中的自我
- **自我 - 他人边界渗透性**: 自我与他人边界的灵活程度

**关系特征**:
- 相互性 (Reciprocity)
- 相互依赖 (Interdependence)
- 共享历史 (Shared History)
- 共同未来预期 (Shared Future)
- 情感联结 (Emotional Bond)

### 4. 群体规范内化 (Norm Internalization) [Cialdini et al. 1990]

**规范焦点理论**:
- **描述性规范** (Descriptive): 群体中大多数人实际做什么
- **指令性规范** (Injunctive): 群体中大多数人认可什么

**内化程度**:
- 外在服从 → 认同接受 → 内在整合

**规范特征**:
- 规范显著性 (Salience)
- 规范强度 (Strength)
- 规范普遍性 (Universality)
- 规范执行机制 (Enforcement)

### 5. 集体自尊 (Collective Self-Esteem) [Luhtanen & Crocker 1992]

**四个维度**:
- **成员资格自尊**: 对自己作为群体成员的评估
- **私人自尊**: 个人对群体的评价
- **公共自尊**: 感知他人对群体的评价
- **重要性自尊**: 群体对自我概念的重要性

**集体自尊水平**:
- 很高 (≥ 0.8)
- 高 (≥ 0.6)
- 中等 (≥ 0.4)
- 低 (≥ 0.2)
- 很低 (< 0.2)

### 6. 认同威胁与应对 [Branscombe et al. 1999]

**威胁类型**:
- **群体价值威胁**: 群体价值受到质疑
- **群体独特性威胁**: 群体独特性受到侵蚀
- **群体成员资格威胁**: 成员身份受到质疑
- **群体控制威胁**: 群体自主性受到限制

**应对策略**:
- 社会创造性 (Social Creativity)
- 社会竞争 (Social Competition)
- 群体差异化 (Group Differentiation)
- 认同肯定 (Identity Affirmation)
- 群体边界强化 (Group Boundary Strengthening)
- 集体行动 (Collective Action)

### 7. 群体情感 (Group-Based Emotion) [Smith & Seger 2009; von Scheve & Salmela 2014]

**核心观点**: 基于群体身份而非个人身份体验的情绪

**群体情感特征**:
- 基于群体身份 (而非个人身份)
- 群体认同调节强度
- 群体规范影响表达
- 群体内共享程度

**集体情绪类型** (von Scheve & Salmela 2014):
- 共享情绪 (Shared Emotion): 高度共享的情绪体验
- 群体情绪 (Group Emotion): 中等共享的情绪体验
- 群体中的个体情绪 (Individual Emotion in Group): 低度共享

### 8. 认同结构与整合

**认同复杂性** (Linville 1985, 1987):
- 简单认同 (1 个认同)
- 中等复杂 (2-3 个认同)
- 复杂认同 (4+ 个认同)

**认同整合度**:
- 高整合：认同之间协调一致
- 中等整合：存在轻微冲突
- 低整合：存在明显冲突

**认同冲突检测**:
- 效价冲突：不同认同的效价相反
- 规范冲突：不同认同的规范不兼容

### 代码使用

```javascript
const { CollectiveIdentityModule } = require('./collective-identity');
const collectiveIdentityModule = new CollectiveIdentityModule();

// 形成社会认同
const identity = collectiveIdentityModule.formSocialIdentity(
  'HeartFlow 开发团队',
  '核心开发者',
  {
    salience: 0.9,
    commitment: 0.95,
    valence: 'positive',
    definingFeatures: ['创新', '开放', '协作', '成长']
  }
);

// 形成认同融合
const fusion = collectiveIdentityModule.formIdentityFusion(
  identity.id,
  0.85, // 高融合度
  {
    sharedHistory: true,
    sharedFuture: true
  }
);

// 形成关系性自我
const relationalSelf = collectiveIdentityModule.formRelationalSelf(
  '开发者伙伴关系',
  ['用户', 'HeartFlow'],
  {
    reciprocity: 0.9,
    interdependence: 0.8,
    sharedHistory: true,
    sharedFuture: true,
    emotionalBond: 0.85,
    permeability: 0.7
  }
);

// 内化群体规范
const norm = collectiveIdentityModule.internalizeGroupNorm(
  'HeartFlow 开发团队',
  '代码审查是必须的',
  'injunctive', // 指令性规范
  {
    salience: 0.8,
    strength: 0.9,
    universality: 0.85,
    enforcement: 'informal'
  }
);

// 体验群体情感
const groupEmotion = collectiveIdentityModule.experienceGroupBasedEmotion(
  'pride', // 自豪
  'HeartFlow 开发团队',
  '项目成功发布 v3.23.0',
  {
    arousal: 0.8,
    identityStrength: 0.9,
    normInfluence: 0.7,
    sharedLevel: 0.85
  }
);

// 计算集体自尊
const cse = collectiveIdentityModule.calculateCollectiveSelfEsteem(identity.id);
console.log('集体自尊水平:', cse.level);
console.log('总体得分:', cse.totalScore.toFixed(2));
console.log('各维度:');
console.log('  成员资格自尊:', cse.dimensions.membershipEsteem.toFixed(2));
console.log('  私人自尊:', cse.dimensions.privateEsteem.toFixed(2));
console.log('  公共自尊:', cse.dimensions.publicEsteem.toFixed(2));
console.log('  重要性自尊:', cse.dimensions.importanceEsteem.toFixed(2));

// 分析认同结构
const analysis = collectiveIdentityModule.analyzeIdentityStructure();
console.log('认同复杂性:', analysis.complexity.level);
console.log('认同整合度:', analysis.integration.level);
console.log('认同冲突:', analysis.conflicts.length);

// 获取状态摘要
const status = collectiveIdentityModule.getStatus();
console.log('社会认同数量:', status.statistics.socialIdentitiesCount);
console.log('认同融合数量:', status.statistics.identityFusionsCount);
console.log('关系性自我数量:', status.statistics.relationalSelvesCount);
```
