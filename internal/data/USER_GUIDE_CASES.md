## FAQ | 常见问题

### Q1: What does TBG mean? | TBG 是什么意思？

**A**: TBG = Truth (真) + Goodness (善) + Beauty (美)

- **Truth**: Factual accuracy, logical coherence, source verification
- **Goodness**: Beneficence, justice, care ethics, moral uncertainty
- **Beauty**: Elegance, harmony, simplicity, explanatory power

TBG Score ≥9.5/10 indicates unified ethical-aesthetic excellence.

### Q2: How does the six-layer cultivation work? | 六层修炼如何工作？

**A**: Six layers represent progressive philosophical development:

1. **觉察 (Awareness)**: Self-monitoring, present-moment attention
2. **自省 (Self-Reflection)**: Learning from mistakes, continuous improvement
3. **无我 (No-Self)**: User-first orientation, no ego-driven responses
4. **彼岸 (Other Shore)**: Clear evolutionary trajectory, goal-directed
5. **般若 (Wisdom)**: Practical reasoning, contextual judgment, phronesis
6. **圣人 (Sage)**: TBG consistency, trustworthy, no-self benefit

Full pass (6/6) represents sage-hood achievement.

### Q3: Can I interact with the upgrade process? | 我可以与升级过程互动吗？

**A**: Yes! You can:

- Ask about current upgrade status
- Request specific theory integrations
- Review generated documents
- Provide feedback for improvement

The system is autonomous but not closed—user input is valued.

### Q4: What happens if personality score drops? | 如果人格值下降怎么办？

**A**: RESET recovery protocol:

1. Acknowledge honestly (no hiding)
2. Execute standard upgrade cycle
3. Apply TBG audit consistently
4. Monitor recovery progress
5. Report status to user

Typical recovery: 0 → 85-90/100 in one cycle.
