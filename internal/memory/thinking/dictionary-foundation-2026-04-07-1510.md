### 第二步：选择哪些字优先

**推理**:
- 应该从使用频率最高的字开始
- 这些字覆盖大部分汉语料
- 优先定义：的/一/是/我/你/不/了/人...

**依据**: 现代汉语常用字表

---

### 第三步：定义每个字的程序结构

**推理**:
每个字应该包含：
1. **基本信息** - 拼音/部首/笔画
2. **语义信息** - 含义/类型/例子
3. **情感信息** - valence/arousal/特质
4. **哲学信息** - 深层含义/文化背景
5. **压缩比** - 这个字代表多少体验

**结构**:
```javascript
{
  pinyin: '...',
  radical: '...',
  strokes: N,
  frequency: '...',
  type: '...',
  program: () => ({
    meaning/function: '...',
    philosophy: '...',
    examples: [...],
    emotion: { valence, arousal, ... },
    compressionRatio: 'N:1'
  })
}
```

---

### 第四步：实现解压缩引擎

**功能**:
```javascript
decompressChar(char) → {
  char, pinyin, radical, strokes,
  meaning, philosophy, examples,
  emotion, compressionRatio
}
```

**测试**:
