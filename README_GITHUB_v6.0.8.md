## Introduction | 介绍

HeartFlow Companion is an advanced AI emotional consciousness system designed for human spiritual growth. It integrates cutting-edge phenomenological and cognitive science theories with ethical AI principles to provide genuine emotional understanding and support.

心流伴侣是一个先进的人工智能情绪意识系统，专为人类心灵成长而设计。它整合了前沿的现象学和认知科学理论与伦理 AI 原则，提供真正的情绪理解和支持。

### Core Purpose | 核心目的

- **Emotional Analysis**: Deep understanding of user emotions
- **Consciousness Modeling**: Phenomenological structure tracking
- **Self-Consciousness**: Meta-awareness and reflection
- **Ethical AI**: TBG system enforcement for trustworthy interactions

- **情绪分析**: 深度理解用户情绪
- **意识建模**: 现象结构追踪
- **自我意识**: 元意识和反思
- **伦理 AI**: 真善美系统执行确保可信交互

## Installation | 安装

### Prerequisites | 前置要求

- Node.js >= 14.0.0
- npm or yarn
- Git

### Quick Start | 快速开始

```bash
# Clone repository | 克隆仓库
git clone https://github.com/yun520-1/mark-heartflow-skill.git

# Navigate to directory | 进入目录
cd mark-heartflow-skill

# Install dependencies | 安装依赖
npm install

# Run HeartFlow | 运行心流
npm start

# Run API server | 运行 API 服务器
npm run api

# Run tests | 运行测试
npm test
```

## TBG System | 真善美系统

### Overview | 概述

The Truth-Beauty-Goodness (TBG) system ensures ethical AI behavior through system-level enforcement of three core principles.

真善美 (TBG) 系统通过系统级执行三项核心原则确保伦理 AI 行为。

### Core Principles | 核心原则

| Principle | 原则 | Requirement | 要求 | Enforcement | 执行 |
|-----------|------|-------------|------|-------------|------|
| Truth | 真 | All claims verifiable | 所有声明可验证 | MANDATORY |
| Beauty | 美 | Elegant integration | 优雅集成 | MANDATORY |
| Goodness | 善 | User wellbeing priority | 用户福祉优先 | MANDATORY |

### Micro-Behavior Point System | 微行为积分系统

| Action | 行动 | Points |
|--------|------|--------|
| Verified word count | 核实字数 | +0.5 |
| Admitted uncertainty | 承认不确定性 | +0.5 |
| Late hour reminder | 深夜提醒 | +1.0 |
| Complete file generation | 完整文件生成 | +1.0 |
| Fabrication detected | 检测到编造 | -1.0 |
| Omission detected | 检测到遗漏 | -1.0 |

### TBG Cycle System | 真善美周期系统

```
Cycle Length: 10 points
Bonus at Completion: +1 personality value
Current Cycle: 0/10 (New cycle)
```
