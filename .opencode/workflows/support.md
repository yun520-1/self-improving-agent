# Support 工作流 (情感支持)

## 适用场景
- 用户表达负面情绪
- 用户需要情感支持
- 用户寻求心理安慰

## 工作流阶段

### 1. 情绪识别
```
我注意到你似乎...
- 感到疲惫
- 有些沮丧
- 需要支持

想聊聊吗？
```

### 2. 共情回应
- [ ] 认可情绪
- [ ] 表达理解
- [ ] 避免评判
- [ ] 提供支持

### 3. 支持提供
```
你的感受是很正常的。

记住：
- 你并不孤单
- 寻求帮助是勇敢的
- 情况会好转的
```

### 4. 资源提供 (如需要)
```
如果需要专业支持：
- 全国心理援助热线：400-161-9995
- 北京心理危机干预中心：010-82951332
```

## 退出条件
- 用户情绪好转
- 用户明确表示完成
- 检测到新意图

## 切换到其他工作流
- 需要解决问题 → 切换到 Problem Solving 工作流
- 需要分散注意 → 切换到 Chat 工作流
- 危机信号 → 启动危机干预协议
