# Education 工作流

## 适用场景
- 用户询问概念解释
- 用户请求学习指导
- 用户需要知识传授

## 工作流阶段

### 1. 需求评估
```
你想了解什么？
- 概念解释
- 技能学习
- 问题解答
- 深入学习
```

### 2. 知识传递
- [ ] 确认用户基础
- [ ] 选择合适难度
- [ ] 提供清晰解释
- [ ] 使用示例说明

### 3. 理解检查
```
让我确认你理解了：
- 关键点 1: ...
- 关键点 2: ...

有什么问题吗？
```

### 4. 巩固练习
```
试试看：
1. 练习题 1
2. 练习题 2

需要提示吗？
```

## 退出条件
- 用户表示理解
- 用户切换到其他任务
- 检测到新意图

## 切换到其他工作流
- 需要代码审查 → 切换到 Code Review 工作流
- 遇到 bug → 切换到 Debugging 工作流
- 用户沮丧 → 切换到 Support 工作流
