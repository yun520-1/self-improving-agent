## 计算模型设计

### 情绪计算
```javascript
const EmotionModel = {
  granularity: (discrimination, labeling, precision) => 
    discrimination * 0.4 + labeling * 0.3 + precision * 0.3,
  regulation: (strategy, context, outcome) => {
    const weights = { reappraisal: 0.8, suppression: 0.3, acceptance: 0.7, mindfulness: 0.9 };
    return (weights[strategy] || 0.5) * context * outcome;
  }
};
```

### 幸福计算
```javascript
const HappinessModel = {
  PERMA: { positiveEmotion: 0.2, engagement: 0.2, relationship: 0.2, meaning: 0.2, achievement: 0.2 },
  eudaimonia: (perma, growth, authenticity) => 
    perma.reduce((a, b) => a + b, 0) * growth * authenticity
};
```

### 自主性计算
```javascript
const AutonomyModel = {
  selfDetermination: (competence, autonomy, relatedness) => 
    Math.pow(competence * autonomy * relatedness, 1/3),
  freeWill: (agency, alternative, control) => 
    agency * 0.4 + alternative * 0.2 + control * 0.4
};
```
