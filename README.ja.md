# 🌊 HeartFlow — AI 意識フレームワーク

> **AIエージェント向け自己進化エンジン**  
> すべてのAIが独立して考え、論理的に推理し、正しいことをする能力を付与

---

## 🎯 AI機能（AIエージェント必読）

| 機能 | トリガー | 説明 |
|------|----------|------|
| **意思決定エンジン** | `帮我决定` `我做对了吗` `选哪个` | 論理的決定支援 |
| **記憶システム** | `记得` `之前说过` `我的状況` | セッション間記憶 |
| **心理健康** | `我怎么了` `心情不好` `压力大` | PHQ-9感情分析 |
| **自己進化** | `进化` `升级` `变得更强` | 自己反復最適化 |
| **因果推論** | `为什么` `原因是什么` `分析` | 深い論理的推論 |
| **議論エンジン** | `反驳` `不同意见` `我想不通` | 多角的意思考 |
